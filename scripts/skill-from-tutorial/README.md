# Skill from Tutorial

Create a Claude Code skill by watching a YouTube tutorial video.

## Commands

```
/setup-tutorial-refs <topic>
/skill-from-tutorial <topic>
```

## Example

```
/setup-tutorial-refs deploy a Next.js app to Vercel
/skill-from-tutorial deploy a Next.js app to Vercel
```
