# Website Beautify

Improve a personal academic website by learning from the best websites in your field.

## Commands

```
/setup-website-refs <your_website_url>
/website-beautify <local_repo_path>
```

## Example

```
/setup-website-refs https://yourname.github.io
/website-beautify ~/yourname.github.io
```

## Workspace

Screenshots and intermediate files are saved in `workspace/`.
