# Logo Design

Design a logo by searching for visual references, then generating with Nano Banana.

## Commands

```
/setup-logo-refs <description>
/logo-design <description>
```

## Example

```
/setup-logo-refs a logo for MMSkills - an open-source toolkit for multimodal agent skills
/logo-design a logo for MMSkills - an open-source toolkit for multimodal agent skills
```

## Workspace

Generated logos and iterations are saved in `workspace/`.
