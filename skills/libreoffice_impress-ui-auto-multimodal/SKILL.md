# LibreOffice Impress 24.2 — UI Reference

LibreOffice Impress is a presentation application (part of the LibreOffice suite) for creating, editing, and presenting slide decks. The default interface features a menu bar, two toolbar rows, a left-hand Slides panel, a central canvas, and a right-hand Properties sidebar.

## Regions

| Region | Description |
|--------|-------------|
| [File Menu](regions/file-menu.md) | Document-level operations: new, open, save, export, print, properties |
| [Edit & View Menus](regions/edit-view-menus.md) | Clipboard, search, undo/redo; view modes, toolbars, panels, zoom |
| [Insert Menu](regions/insert-menu.md) | Adding content: images, tables, charts, shapes, media, hyperlinks, fields |
| [Format Menu](regions/format-menu.md) | Text styling, object transforms, alignment, arrangement, grouping |
| [Slide & Slide Show Menus](regions/slide-slideshow-menus.md) | Slide management, presentation playback, plus Tools/Window/Help |
| [Standard Toolbar](regions/standard-toolbar.md) | First icon row — file ops, clipboard, undo, search, view, presentation, insert |
| [Drawing Toolbar](regions/drawing-toolbar.md) | Second icon row — selection, shapes, colors, rotation, alignment |
| [Slides Panel](regions/slides-panel.md) | Left-side slide thumbnail list with reorder and context menu |
| [Properties Sidebar](regions/properties-sidebar.md) | Right-side panel: slide properties, layouts, and sidebar tab strip |
| [Canvas & Status Bar](regions/canvas-statusbar.md) | Main editing surface and bottom status/zoom bar |
| [PDF Options Dialog](regions/pdf-options-dialog.md) | 6-tab export dialog for PDF output |
| [Character & Paragraph Dialogs](regions/character-paragraph-dialogs.md) | Multi-tab dialogs for text and paragraph formatting |
