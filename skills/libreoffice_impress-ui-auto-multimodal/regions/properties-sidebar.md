# Properties Sidebar

The right-hand sidebar contains the Properties panel (slide settings and layouts) and a vertical tab strip for switching between sidebar panels.

## Screenshot

![overall](../images/properties-sidebar-overall.png)

## Slide Section

| Control | Description |
|---------|-------------|
| Format dropdown | Page/slide format presets: On-screen Show (4:3/16:9/16:10), paper sizes (A0–A5, Letter, Legal, etc.), Screen ratios, Dia Slide, Japanese Postcard, User |
| Orientation dropdown | Landscape (default) or Portrait |
| Background dropdown | None, Color, Gradient, Hatching, Bitmap, Pattern, Use Slide Background |
| Insert Image button | Opens file picker to set a slide background image (distinct from fill-type backgrounds above) |
| Master Slide dropdown | Select which master slide to apply (e.g., "Default") |
| Master Background checkbox | Show/hide master slide's background (default: on) |
| Master Objects checkbox | Show/hide objects placed on the master slide (default: on) |
| Master View button | Switches to master slide edit mode. Button changes to "Close Master View" to return to normal editing. |

## Layouts Grid

A 3x4 grid of 12 clickable layout thumbnails. Click to apply the layout to the current slide.

Row 1: Blank Slide, Title Slide, Title+Content, Title+2 Content
Row 2: Title Only, Centered Text, Title+2 Content and Content, Title+Content and 2 Content
Row 3: Title+2 Content over Content, Title+Content over Content, Title+4 Content, Title+6 Content

## Sidebar Tab Strip

Eight icon buttons on the far-right edge, each toggling a sidebar panel:

| Icon | Panel | Shortcut |
|------|-------|----------|
| 1 | Properties | Alt+1 |
| 2 | Styles | Alt+2 |
| 3 | Gallery | Alt+3 |
| 4 | Navigator | Alt+4 |
| 5 | Shapes | Alt+5 |
| 6 | Slide Transition | Alt+6 |
| 7 | Animation | Alt+7 |
| 8 | Master Slides | Alt+8 |
