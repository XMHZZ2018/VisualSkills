# Selecting the User Interface (LibreOffice Writer 7.3.7)

LibreOffice Writer ships with the classic menu-and-toolbar layout, but it also offers several alternative interface styles that group commands differently. You can switch between them at any time — nothing is permanent.

To get started, go to **View > User Interface** on the menu bar. This opens the **Select Your Preferred User Interface** dialog, where a list of variants appears on the left and a live preview on the right so you can see what you're choosing before you commit.

See `fig01.png`.

Pick the variant you want, then decide the scope: hit **Apply to All** to change every LibreOffice application at once, or **Apply to Writer** to keep the change local. Click **Close** when you're done, and the Writer window reshapes itself immediately.

Here's what each of the standard variants gives you:

**Standard Toolbar** is the classic look — two visible toolbars (Standard and Formatting) with the sidebar showing only its tabs. If you've been using LibreOffice for years, this is probably what you're used to.

**Single Toolbar** strips things down to one toolbar carrying the most frequently used features. The sidebar shows only the Hide button, staying out of your way until you need it.

**Sidebar** goes the other direction — it opens the sidebar fully (with the Formatting panel visible) and keeps just one toolbar on screen.

**Tabbed** mimics the ribbon-style interface found in other office suites, replacing traditional menus and toolbars with a row of contextual tabs.

**Tabbed Compact** is the same idea as Tabbed but squeezed into less vertical space — handy on smaller screens.

**Groupedbar Compact** and **Contextual Single** offer further compact layouts that organize tools into grouped or context-sensitive panels.

If you've enabled experimental features under **Tools > Options > Advanced**, a few additional variants will appear on the left side of the dialog. Those are still evolving, so stick with the standard choices unless you're feeling adventurous.
