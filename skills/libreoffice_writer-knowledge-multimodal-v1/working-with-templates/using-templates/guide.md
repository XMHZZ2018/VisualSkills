# Creating Documents from Templates (LibreOffice Writer 7.3.7)

A template is basically a model document — it holds the layout, styles, graphics, and other setup so every new document you create from it starts out looking consistent. Think company letterheads, business reports, or anything you want to reuse without rebuilding from scratch. Templates can also bundle things like custom measurement units, default printer settings, and toolbar configurations.

Every document in LibreOffice is technically based on a template. If you never pick one explicitly, Writer just uses its built-in blank template behind the scenes. You can create, download, or install as many templates as you like, and even set a different default for each document type (text, spreadsheet, drawing, presentation).

To create a new document from a template, you first need to open the Templates dialog. There are several ways to get there — pick whichever feels natural. You can go to **File > New > Templates** or **File > Templates > Manage Templates** from the menu bar. The keyboard shortcut **Ctrl+Shift+N** works too. You can also click the arrow next to the **New** icon on the Standard toolbar and choose **Templates**. If you're on the Start Center with no document open, just select **Templates > Manage Templates** from there.
Once the dialog is open, use the **Filter** tabs at the top to pick the type and category of template you're after. The relevant templates will appear in the main area. You can switch between a thumbnail view and a list view using the icons next to the **Categories** list — thumbnail view is handy for visually browsing, while list view is better when you know what you're looking for.

When you spot the template you want, just double-click it, or select it and click **Open**. Writer creates a brand-new document based on that template, ready for you to edit and save. The template the document is based on gets recorded in **File > Properties > General**, so you can always check later.

One quick tip: from the Start Center, you can also click the **Templates** button or use its drop-down menu to pick a template type directly — this lets you browse and create a new document without even opening the full Templates dialog.
