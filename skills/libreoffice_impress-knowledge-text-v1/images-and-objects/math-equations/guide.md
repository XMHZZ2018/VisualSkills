# Math Equation Objects (LibreOffice Impress 7.3.7)

Impress lets you drop mathematical formulas right into your slides using the built-in LibreOffice Math editor. There are two ways to get a formula onto a slide: create one from scratch via the formula editor, or insert it as an OLE object.

The quickest route is **Insert > Object > Formula Object** on the Menu bar. This opens the Math editor directly, where you can type markup to build your equation. While editing, the **Math** menu appears in the Menu bar, giving you access to all of Math's symbol and structure tools.

Alternatively, go to **Insert > Object > OLE Object**, select **Create new**, choose **LibreOffice 7.3 Formula** from the **Object Type** list, and click **OK**. The formula object lands in the center of the slide in edit mode with the Math toolbars ready to go. If you already have a formula saved as a file, select **Create from file** instead, use **Search** to locate it, then click **OK** to embed it.

Keep an eye on font sizing — formulas should roughly match the text on your slide. Adjust formula font size through **Format > Font Size** and change the typeface via **Format > Fonts**, both on the Menu bar while the formula is in edit mode.

One thing to be aware of: unlike in Writer, a formula in Impress is treated as a standalone object. It won't automatically align with surrounding text or other objects on the slide. You can freely move the formula around, but you cannot resize it by dragging its handles — font size is the only way to control its dimensions.

Double-click any formula object on a slide to jump back into the Math editor and make changes. Click outside the object when you're done to return to normal slide editing.
