# Add Speaker Notes (LibreOffice Impress 7.3.7)

Notes provide reminders or extra information for slides and are not displayed during a presentation. With dual displays, notes can be shown as presentation cues on a second screen.

## Adding Notes

1. Select a slide to add notes to, then open Notes view using one of:
   - Click **Notes** at the top of the Workspace
   - Go to **View > Notes** on the Menu bar
   - Click **Display Views** and select **Notes** in Edit Modes on the Standard toolbar

   Read the screenshot `fig02.png` in this directory — it shows the Notes view in the Workspace (Figure 246).

2. Click on **Click to add notes** below the slide, then type or paste text or graphics into the notes box.
3. To add notes to another slide, repeat Steps 1 and 2.
4. When finished, return to Normal view.

## Formatting Notes (via Master Notes)

It is recommended to use Master Notes to format notes appearance rather than formatting individually per slide.

1. Select a slide that has notes added.
2. Open **Master Notes** view using one of:
   - Go to **View > Master Notes** on the Menu bar
   - Click **Master** on the Standard toolbar, then click **Notes** at the top of the Workspace
   - Click **Display Views** and select **Master Notes** in Master Modes on the Standard toolbar
3. Open the **Slide Properties** dialog using one of:
   - Go to **Slide > Slide Properties** on the Menu bar
   - Right-click in an empty area on the master note and select **Slide Properties**

   Read the screenshot `fig04.png` in this directory — it shows the Slide Properties dialog in Notes view (Figure 248).

4. Use the options in the Slide Properties dialog to set up the master note, then click **OK**.

## Modifying the Notes Text Style

1. Click **Styles** on the Sidebar to open the Styles deck.
2. Click **Presentation Styles** at the top of the Styles deck.
3. Right-click on **Notes** presentation style and select **Modify**. This opens the Notes dialog.
4. Use the pages and options in the Notes dialog to modify the Notes style, then click **OK**.

## Headers and Footers in Notes

1. Open Notes view, then open Master Notes view (see steps above).
2. Go to **Insert > Header and Footer** on the Menu bar.
3. Click **Notes and Handouts** to open that page.

   Read the screenshot `fig05.png` in this directory — it shows the Header and Footer dialog on the Notes and Handouts page (Figure 250).

4. Select **Header** and enter header text as required.
5. Select **Date and time** and choose Fixed (enter date/time manually) or Variable (choose format and language from drop-downs).
6. Select **Footer** and enter footer text as required.
7. Select **Page Number** to show page numbers on each notes page.
8. Click **Apply to All**.

## Printing Notes

**Linux or Windows:**
1. Open Notes view, then go to **File > Print** (or **Ctrl+P**).
2. Select the **LibreOffice Impress** tab.
3. In **Document**, select **Notes** from the Type drop-down list.
4. Select any additional options (e.g. Slide name, Original colors, Original size), then click **OK**.

Read the screenshot `fig07.png` in this directory — it shows the Print dialog LibreOffice Impress page for Linux or Windows (Figure 251).

**macOS:**
1. Open Notes view, then go to **File > Print** (or **⌘P**).
2. Make sure **LibreOffice** is selected from the drop-down list.
3. In **LibreOffice Impress**, select **Notes** from the Document drop-down list.
4. Select any additional options, then click **OK**.

## Exporting Notes as PDF

The exported PDF contains separate pages for slides and notes (e.g. 10 slides = 20 PDF pages).

1. Open Notes view, then go to **File > Export As > Export as PDF**.
2. On the **General** page, select **Export notes pages**.

   Read the screenshot `fig08.png` in this directory — it shows the PDF Options dialog General page (Figure 253).

3. Optionally select **Export only notes pages** to create a PDF with notes only.
4. Configure other pages in the PDF Options dialog as needed, then click **Export**.
5. Navigate to the save folder, enter a filename, and click **Save**.
