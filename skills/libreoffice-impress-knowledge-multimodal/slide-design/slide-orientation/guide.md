# Change Slide Orientation (LibreOffice Impress 7.3.7)

Slide orientation can be changed using either the Slide Properties dialog or the Sidebar.

---

## Method 1: Slide Properties Dialog

1. Make sure that Normal view is selected in the Workspace.
2. Open the Slide Properties dialog using one of the following methods:
   - Go to **Slide > Slide Properties** on the Menu bar.
   - Right-click in an empty area on the slide and select **Slide Properties** from the context menu.
3. Read the screenshot `fig02.png` in this directory — it shows the Slide Properties dialog (Slide page) where orientation and size options are available.
4. Use the options on the **Slide** page to set the orientation.
5. Click **OK** to save the changes and close the dialog.

---

## Method 2: Sidebar Slide Panel

1. Make sure that Normal view is selected in the Workspace.
2. Click on **Properties** in the Sidebar to open the Properties deck.
3. Click on **Slide** to open the Slide panel in the Properties deck.
4. Read the screenshot `fig03.png` in this directory — it shows the Slide panel in the Properties deck on the Sidebar.
5. Select the slide orientation to use from the **Orientation** drop-down list.

Options selected in the Sidebar Slide panel are applied immediately to the slide.
