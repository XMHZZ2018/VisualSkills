# Set Slide Background Color (LibreOffice Impress 7.3.7)

## Method 1: Via Slide Properties Dialog (Normal View)

1. Make sure Normal view is selected in the Workspace.
2. Open the Slide Properties dialog using one of these methods:
   - Go to **Slide > Slide Properties** on the Menu bar.
   - Right-click in an empty area on the slide and select **Slide Properties** from the context menu.
3. Select the **Background** tab to open the Background page. Read the screenshot `fig12.png` in this directory — it shows the Slide Properties dialog Background page.
4. Select the type of background required from: **Color**, **Gradient**, **Bitmap**, **Pattern**, or **Hatch**.
5. Click **OK** to save the changes and close the dialog.

## Method 2: Via Master Slide and Slide Properties

1. Go to **View > Master Slide** on the Menu bar.
2. Select a master slide in the Slides pane so that it appears in the Workspace.
3. Go to **Slide > Slide Properties** on the Menu bar, or right-click in the Workspace and select **Slide Properties** from the context menu.
4. Select the **Background** tab.
5. Select the type of background required from: **Color**, **Gradient**, **Bitmap**, **Pattern**, or **Hatch**.
6. Click **OK** to save the changes and close the dialog.

## Method 3: Via Presentation Styles (Master Slide)

1. Go to **View > Master Slide** on the Menu bar.
2. Select a master slide in the Slides pane so that it appears in the Workspace.
3. Click **Styles** on the Sidebar to open the Styles deck.
4. Click **Presentation Styles** to open the Presentation Styles list. Read the screenshot `fig01.png` in this directory — it shows Presentation Styles in the Styles deck on the Sidebar.
5. Right-click on **Background** style and select **Modify** from the context menu to open the Background dialog.
6. Select the type of background required from: **Color**, **Gradient**, **Bitmap**, **Pattern**, or **Hatch**.
7. Click **OK** to save the changes and close the dialog.

## Method 4: Via Sidebar Slide Panel (Normal View)

1. Make sure Normal view is selected in the Workspace.
2. Click **Properties** in the Sidebar to open the Properties deck.
3. Click **Slide** to open the Slide panel. Read the screenshot `fig10.png` in this directory — it shows the Slide panel in the Properties deck on the Sidebar.
4. Select the slide background to use from the **Background** drop-down list. The background type selected changes the options available.
5. Changes are applied immediately to the slide.
