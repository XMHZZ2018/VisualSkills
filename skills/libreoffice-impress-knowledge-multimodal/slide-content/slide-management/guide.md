# Manage Slides (Duplicate, Summary) (LibreOffice Impress 7.3.7)

## Duplicating Slides

Duplicating a slide copies its formatting, layout, and animations. The duplicate is inserted after the selected slide.

1. Open Normal or Slide Sorter view in the Workspace.
2. Select the slide(s) for duplication in the Slides pane or Slide Sorter view.
3. Use one of the following methods:
   - Go to **Slide > Duplicate Slide** on the Menu bar.
   - Right-click on the selected slide in the Slides pane and select **Duplicate Slide**.
   - Right-click in Slide Sorter view and select **Duplicate Slide**.
   - Click **Duplicate Slide** on the Presentation toolbar.

**Tip:** If a slide becomes crowded with information, duplicate it and split the content across two or more slides. All formatting, backgrounds, and animations are preserved.

## Expanding Slides

Use Expand Slide when a slide has too many points to fit in the available space.

1. Select the slide for expansion in the Slides pane.
2. Go to **Slide > Expand Slide** on the Menu bar. New slides are created from the first outline level; the first outline level becomes the title of each new slide, and lower outline levels move up one level.
3. If necessary, rearrange the new slides in the presentation order.

Read the screenshot `fig09.png` in this directory — it shows an example of expanded slides (Figure 234).

**Note:** The selected slide must contain an outline for Expand Slide to work.

## Creating Summary Slides

A summary slide is useful for creating an agenda. Bullet points are generated from the titles of slides in the presentation.

1. Select the slide that will be the first slide to appear in the summary.
2. Go to **Slide > Summary Slide** on the Menu bar. A summary slide is created at the end of the presentation.
3. If necessary, move the summary slide to the desired position in the presentation.

Read the screenshot `fig10.png` in this directory — it shows an example of a summary slide (Figure 235).

**Note:** The selected slide must contain an outline for Summary Slide to work.

## Creating Slides from a Writer Outline

### Using a Writer Document
1. Open the Writer file in LibreOffice Writer.
2. Go to **File > Send > Outline to Presentation** on the Writer Menu bar. A new Impress presentation opens in Outline view; heading paragraph styles are converted to Impress outline styles.
3. If necessary, duplicate or expand slides and manually adjust slide contents.

Read the screenshot `fig11.png` in this directory — it shows an example of an outline created from a Writer document (Figure 236).

### Using AutoAbstract
1. Open the Writer file in LibreOffice Writer.
2. Go to **File > Send > AutoAbstract to Presentation** on the Writer Menu bar to open the Create AutoAbstract dialog.
3. Select the number of outline levels in the **Included outline levels** box.
4. Select the number of paragraphs per level in the **Paragraphs per level** box.
5. Click **OK** to create the outline in a new Impress presentation.
6. If necessary, duplicate or expand slides and manually adjust contents.
