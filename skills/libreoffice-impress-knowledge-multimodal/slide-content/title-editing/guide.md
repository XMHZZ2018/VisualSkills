# Edit Slide Title and Text Boxes (LibreOffice Impress 7.3.7)

## Adding Text to a Slide

Two methods exist for adding text:

- **Contents box**: Click "Click to add Title" or "Click to add Text" in an AutoLayout text box. Impress enters text editing mode and the Text Formatting toolbar opens automatically.
- **Text box**: Click Insert Text Box on the Standard or Drawing toolbar (Read the screenshot `fig05.png` in this directory — it shows the Standard toolbar with Insert Text Box), then click on the slide. Type text, then click outside to end text mode. Use F2 as a keyboard shortcut for horizontal text boxes.

For vertical text boxes, use Insert Vertical Text (requires Asian/CTL options enabled via Tools > Options > Language Settings > Languages).

## Using AutoLayout Text Boxes

1. In Normal view, select a slide layout from the Layouts panel in the Properties deck on the Sidebar (Read the screenshot `fig06.png` in this directory — it shows the Layouts panel in the Properties deck on Sidebar).
2. Click "Click to add Title" or "Click to add Text" — the placeholder text disappears and a text cursor appears.
3. Type or paste text into the box.
4. Click outside the text box to deselect it.

## Moving a Text Box

1. Click in the text box so the border is displayed.
2. Move the cursor over the border until it changes to the move cursor (e.g., a hand).
3. Click, hold, and drag to the new position. A dashed outline shows the destination (Read the screenshot `fig10.png` in this directory — it shows a text box being moved with a dashed outline).
4. Release the button, then click outside to exit edit mode.

## Resizing a Text Box

1. Click in the text box so the selection handles are displayed.
2. Move the cursor over a selection handle until the cursor changes shape.
3. Click, hold, and drag a handle to resize:
   - Corner handles change width and height simultaneously.
   - Top/bottom handles change height only.
   - Left/right handles change width only.
4. Release, then click outside to exit edit mode.

## Removing a Text Box or Element

1. Click the contents box or element to highlight it (selection handles appear).
2. Press Delete (macOS: Ctrl+D) or Backspace.

## Adding a Border to a Text Box

**Via Line and Filling toolbar** (Read the screenshot `fig07.png` in this directory — it shows the Line and Filling toolbar):
1. Select the text box.
2. In Line Style, select a style from the drop-down list.
3. In Line Width, enter a width value.
4. In Line Color, select a color.
5. Click outside to exit edit mode.

**Via Line dialog**:
1. Select the text box, then right-click the border and select Line, or go to Format > Object and Shape > Line.
2. On the Line page (Read the screenshot `fig08.png` in this directory — it shows the Line dialog Line page), set Style, Color, Width, Transparency, and Corner Style.
3. Click OK, then click outside to exit edit mode.
