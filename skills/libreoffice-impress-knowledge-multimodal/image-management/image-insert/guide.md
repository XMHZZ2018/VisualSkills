# Insert an Image (LibreOffice Impress 7.3.7)

## Inserting an Image from a File

1. Insert a new slide or select an existing slide in the presentation.
2. If necessary, select a layout for the slide.
3. Go to **Insert > Image** on the Menu bar — the Insert Image file browser opens. Read the screenshot `fig01.png` in this directory — it shows Figure 98: an example of the Insert Image file browser.
4. Navigate to the directory containing the required image and select the file.
5. If necessary, select **Preview** to display a thumbnail in the preview pane on the right.
6. Click **Open** to place the image in the center of the slide. The image appears with selection handles around its border.
7. Format the image as needed (see "Formatting images").

### Linking Image Files
To insert a linked image instead of embedding it, select **Insert as Link** in the Insert Image file browser. Linking is useful when:
- The image file is large and embedding would increase the presentation file size significantly.
- The same image file is used across many presentations.
- The linked file is only available when the presentation is opened.

## Inserting a Scanned Image

1. Prepare the image and ensure the scanner is switched on.
2. If the scanner has not been used before, go to **Insert > Media > Scan > Select Source** on the Menu bar.
3. If the scanner is already configured, go to **Insert > Media > Scan > Request** on the Menu bar.
4. If multiple scanners are connected, select the scanner from the **Select Source** or **Device Used** drop-down list.
5. If available, click **Create Preview** or **Preview** to generate a preview scan.
6. If necessary, crop the image and make scanning adjustments.
7. Click **Scan** — Impress places the scanned image in the center of the selected slide.
8. Format the image as needed.

## Inserting an Image from the Gallery

1. Open the Gallery deck in the Sidebar using one of:
   - **Insert > Media > Gallery** on the Menu bar.
   - Click **Gallery** in the Sidebar.
   Read the screenshot `fig03.png` in this directory — it shows Figure 99: the Gallery deck on the Sidebar.
2. Select a theme from the available themes — its images are displayed.
3. Scroll through the images and select the required one.
4. Click, hold, and drag the image onto the slide. Release to insert it centrally.
5. Format the image as needed.

### Creating a Custom Theme
1. Open the Gallery deck and click **New** at the bottom — the Properties of New Theme dialog opens. Read the screenshot `fig04.png` in this directory — it shows Figure 100: the Properties of New Theme dialog, General page.
2. Click **General** and type a name for the new theme.
3. Click **OK** to save and close. The new theme appears in the theme list.

### Adding Images to a Custom Theme
1. Right-click a custom theme name and select **Properties** from the context menu.
2. Click **Files**, then click **Find Files** to open a Select Path dialog. Read the screenshot `fig05.png` in this directory — it shows Figure 101: the Properties of New Theme dialog, Files page.
3. Navigate to the folder containing the required images and select it.
4. Click **OK** — the files appear in the dialog.
5. Select the required files and click **Add** (or **Add All** for all files).
6. Click **OK** to close the dialog.

### Deleting Images from a Custom Theme
1. Open the Gallery deck and select the custom theme.
2. Right-click the image and select **Delete**.
3. Click **Yes** in the confirmation message. Note: only the Gallery link is deleted; the original file is not affected.
