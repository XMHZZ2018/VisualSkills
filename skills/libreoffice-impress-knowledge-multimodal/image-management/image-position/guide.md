# Move / Reposition an Image (LibreOffice Impress 7.3.7)

## Method 1: Using the Cursor

1. Click on the image to display the selection handles.
2. Move the cursor over the image until the cursor changes shape.
3. Click and drag the image to the desired position. Read the screenshot `fig01.png` in this directory — it shows a ghost image of the object appearing during movement to help with repositioning.
4. Release the cursor when the image is in the desired position.

> The arrow keys can also be used to quickly move a selected object to a new position.

---

## Method 2: Position and Size Dialog (Precise Placement)

1. Click the image to select it, then open the Position and Size dialog using one of:
   - Keyboard shortcut **F4**
   - **Format > Object and Shape > Position and Size** on the Menu bar
   - Right-click the image and select **Position and Size**
   - Click **More Options** on the right of the Position and Size panel title bar in the Sidebar Properties deck
2. Click the **Position and Size** tab.
3. In the **Position** section, specify **Position X** (horizontal) and **Position Y** (vertical). Values are measured from the selected **Base point** (default: top-left corner of slide). Read the screenshot `fig03.png` in this directory — it shows the Position and Size dialog with the Position and Size page open.
4. To lock the position, select the **Position** option in the **Protect** section.
5. Click **OK**.

> Units of measurement are set in **Tools > Options > LibreOffice Impress > General** (macOS: **LibreOffice > Preferences > LibreOffice Impress > General**).

---

## Method 3: Sidebar Position and Size Panel

1. Click the image to select it.
2. Click **Properties** in the Sidebar to open the Properties deck.
3. Click **Position and Size** to open the panel. Read the screenshot `fig04.png` in this directory — it shows the Position and Size panel in the Properties deck on the Sidebar.
4. Specify **Position X** (horizontal) and **Position Y** (vertical). Values are relative to the Base point set in the Position and Size dialog.
5. Press **Enter** to apply the changes.
