# Resize an Image (LibreOffice Impress 7.3.7)

## Method 1: Drag with Cursor

1. Click on the image to select it and display the selection handles.
2. Move the cursor over one of the selection handles. The cursor changes shape to indicate the direction of resizing.
3. Click, hold and drag the selection handle to resize the image. A ghost image appears during resizing. Read the screenshot `fig07.png` in this directory — it shows an example of resizing objects with a ghost image.
4. To maintain the width-to-height ratio, hold down the Shift key before clicking and dragging a selection handle. Release the cursor before releasing the Shift key.
5. Release the cursor when the object is at the desired size.

**Note:** Corner handles resize both width and height simultaneously. Top/bottom handles resize height only. Left/right handles resize width only.

---

## Method 2: Position and Size Dialog (Precise)

1. Select the image to display the selection handles, then open the Position and Size dialog using one of:
   - Keyboard shortcut **F4**
   - **Format > Object and Shape > Position and Size** on the Menu bar
   - Right-click the image and select **Position and Size**
   - Click **More Options** in the Position and Size panel on the Sidebar
2. Click the **Position and Size** tab.
3. In the **Size** section, select a **Base point** to anchor during resizing (default: top left corner stays fixed).
4. To maintain proportions, select **Keep ratio** before changing Width or Height — changes to one dimension automatically adjust the other.
5. Specify the **Width** and **Height** values. Read the screenshot `fig08.png` in this directory — it shows the Position and Size dialog with the Size section and Keep ratio option.
6. Click **OK** to save and close the dialog.

---

## Method 3: Sidebar Position and Size Panel

1. Select the image to display the selection handles.
2. Click **Properties** in the Sidebar to open the Properties deck.
3. Click **Position and Size** to open the panel. Read the screenshot `fig09.png` in this directory — it shows the Position and Size panel in the Properties deck on the Sidebar.
4. To maintain proportions, select **Keep ratio** before changing Width or Height.
5. Enter the desired **Width** and **Height**, then press **Enter**.

---

**Tip:** Resizing a raster (bitmap) image affects resolution and may cause blurring. For best results, scale the image to the desired size in a graphics program (e.g., GIMP) before inserting it into a presentation.
