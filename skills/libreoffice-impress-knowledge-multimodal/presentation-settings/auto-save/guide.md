# Configure Auto-Save (LibreOffice Impress 7.3.7)

## Overview

LibreOffice options apply across all modules, including Impress. Settings are accessed via the Options LibreOffice dialog. This chapter covers key option pages available under **Tools > Options** (macOS: **LibreOffice > Preferences**).

---

## Accessing the Options Dialog

1. Go to **Tools > Options** (macOS: **LibreOffice > Preferences**) on the Menu bar to open the Options LibreOffice dialog.
2. The list on the left-hand side varies depending on which component of LibreOffice is open.
3. Click the **expansion marker** next to the relevant section on the left to open subsections.
4. Select the desired options page, make changes, then click **OK** to save and close.

> Note: **Revert** resets all pages in the Options dialog to the values in place when LibreOffice was opened.

---

## Impress-Specific Options

To access Impress options:

1. Go to **Tools > Options** on the Menu bar.
2. Click the **expansion marker** next to **LibreOffice Impress** on the left-hand side.

Read the screenshot `fig06.png` in this directory — it shows the Options LibreOffice Impress dialog General page, listing available sections.

3. Select **General** to specify general options for slide shows.
4. Click **OK** to save changes and close the dialog.

Key General settings include:
- **Start with Template Selection** — opens the template dialog when creating a new presentation.
- **Copy when moving** — creates a copy when moving an object while holding **Ctrl**.
- **Use background cache** — speeds up display of master slide objects.
