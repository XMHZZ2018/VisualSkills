# Presenter View / Monitor Setup (LibreOffice Impress 7.3.7)

## Slide Show Settings

1. Go to **View > Slide Sorter** on the Menu bar. Read the screenshot `fig01.png` in this directory — it shows the Slide Sorter view.
2. Go to **Slide Show > Slide Show Settings** on the Menu bar to open the Slide Show Settings dialog. Read the screenshot `fig02.png` in this directory — it shows the Slide Show Settings dialog.

## Multiple Displays (Presentation Display)

Under **Multiple Displays** in the Slide Show Settings dialog:
- **Presentation display** — select from the drop-down list which display to use for full screen slide show mode. Only available if the computer is connected to more than one display monitor.

> **Note:** The multiple displays setting is not saved with the presentation file. It is saved as a local setting in the user configuration for the computer. If the presentation is opened on a different computer, that computer's local settings are applied.

## Presenter Console

The Presenter Console provides extra control over slide shows when using dual displays (e.g., a laptop for control and a projector for the audience). The laptop display shows the current slide, next slide, slide notes, and a presentation timer.

> **Note:** The Presenter Console only works with operating systems that support two displays.

The Presenter Console provides the following views:

- **Default view** — displays the current slide (including effects) and the next slide. Use Previous and Next arrows to navigate. Read the screenshot `fig07.png` in this directory — it shows the Presenter Console default view.
- **Notes view** — click **Notes** to display notes accompanying each slide. Click **Notes** again to return to the default view.
- **Slides view** — click **Slides** to display slide thumbnails. Click **Slides** again to return to the default view. Read the screenshot `fig08.png` in this directory — it shows the Presenter Console Slides view.
- **Exchange** — click this icon to switch the Presenter Console between displays.
