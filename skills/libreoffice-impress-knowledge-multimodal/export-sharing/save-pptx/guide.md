# Save as PowerPoint Format

1. Open your presentation in LibreOffice Impress.
2. Go to File > Save As to open the Save As dialog.
3. In the Save As dialog, click the file type dropdown (labeled 'All Formats' or showing the current format) and select 'PowerPoint 2007-2019 (.pptx)'.

   ![Step 3](step03.png)

4. Enter a filename if needed, then click Save.
5. A warning dialog will appear asking about the format. Click 'Use Microsoft PowerPoint Format!' to confirm saving in .pptx format.

   ![Step 5](step05.png)
