# Insert a Table (LibreOffice Impress 7.3.7)

## Overview
Tables can be created directly in a presentation. When inserted, a table is placed inside a text box at the center of the slide with default style and settings applied.

## Method 1: Insert Table Dialog
1. Select the slide that will contain the table and modify the slide layout if needed to create space.
2. Go to **Insert > Table** on the Menu bar to open the Insert Table dialog.
   Read the screenshot `fig02.png` in this directory — it shows the Insert Table dialog (Figure 82).
3. Specify the **Number of columns** and **Number of rows** in the text boxes.
4. Click **OK** to insert the table and close the dialog.

## Method 2: Table Graphic
1. Select the slide that will contain the table and modify the slide layout if needed.
2. Click **Table** on the Standard or Table toolbar to open the Table graphic.
   Read the screenshot `fig03.png` in this directory — it shows the Table graphic (Figure 83).
3. Click, hold, and drag the cursor across and down in the table graphic to create the required number of rows and columns.
4. Release the cursor to insert the table and close the Table graphic.
5. Alternatively, click **More Options** in the Table graphic to open the Insert Table dialog.

## After Insertion
- The **Table toolbar** opens automatically when a table is selected. If not visible, go to **View > Toolbars > Table**.
  Read the screenshot `fig02.png` in this directory — it shows the Table toolbar (Figure 81).
- To apply a design, open the **Table Design** panel in the Properties deck on the Sidebar.
  Read the screenshot `fig01.png` in this directory — it shows the Table Design panel (Figure 80).
- Table Design options include: Header row, Total row, Banded rows, First column, Last column, Banded columns.
  Read the screenshot `fig04.png` in this directory — it shows an example table with banded rows and columns (Figure 84).
