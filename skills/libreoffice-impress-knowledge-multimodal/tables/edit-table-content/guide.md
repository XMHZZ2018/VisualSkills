# Edit Table Content (LibreOffice Impress 7.3.7)

## Inserting a Table

**Using the Insert Table dialog:**
1. Select the slide that will contain the table; modify the slide layout if needed.
2. Go to **Insert > Table** on the Menu bar to open the Insert Table dialog.
3. Specify the **Number of columns** and **Number of rows**.
4. Click **OK** to insert the table.

**Using the Table graphic:**
1. Click **Table** on the Standard or Table toolbar to open the Table graphic. Read the screenshot `fig03.png` in this directory — it shows the Table graphic for selecting rows and columns.
2. Click, hold, and drag across and down to select the required number of rows and columns.
3. Release the cursor to insert the table, or click **More Options** to open the Insert Table dialog instead.

> Note: Tables are created inside a text box at the center of the slide using default style and settings.

## Formatting Tables

### Sidebar Table Design Panel
1. Insert or select a table, then click **Table Design** in the Properties deck on the Sidebar. Read the screenshot `fig01.png` in this directory — it shows the Table Design panel.
2. Select a design and configure row/column options:
   - **Header row** – first row displayed with a different background (selected by default).
   - **Total row** – highlights the last row.
   - **Banded rows** – alternate row backgrounds for readability (selected by default).
   - **First column** / **Last column** – highlights the respective column with a darker background.
   - **Banded columns** – alternate column highlights. Read the screenshot `fig04.png` in this directory — it shows an example table with banded rows and columns.

### Table Toolbar
When a table is selected, the Table toolbar appears automatically. Read the screenshot `fig02.png` in this directory — it shows the Table toolbar. Key tools include:

- **Border Style / Border Color / Borders** – apply border styles and colors to selected cells.
- **Area Style/Filling** – apply None, Color, Gradient, Hatching, Bitmap, or Pattern fill to selected cells.
- **Merge Cells** – merges selected cells. Also via right-click > **Merge Cells** or **Format > Table > Merge Cells**.
- **Split Cells** – splits a cell horizontally or vertically. Click the cell, then click **Split Cells** to open the Split Cells dialog. Read the screenshot `fig05.png` in this directory — it shows the Split Cells dialog. Select number of cells and direction; use **Into equal proportions** for equal horizontal splits.
- **Optimize** – evenly distributes rows/columns. Opens the Optimize toolbar with: Minimal/Optimal Column Width, Minimal/Optimal Row Height, Distribute Columns Evenly, Distribute Rows Equally.
- **Align Top / Center Vertically / Align Bottom** – sets vertical text alignment. Also via **Format > Align Text**.
- **Insert Row Above/Below, Insert Column Before/After** – insert rows or columns. Also via **Format > Table > Insert Rows/Columns**. Read the screenshot `fig06.png` in this directory — it shows the Insert Rows and Insert Columns dialogs.
- **Delete Row / Delete Column / Delete Table** – delete selected rows, columns, or the full table. Also via **Format > Table** submenu.

### Table Properties Dialog
Open via right-click > **Table Properties**, **Format > Table > Properties**, or **Table Properties** on the Table toolbar.

- **Font** – set Family, Style, Size, and Language. Read the screenshot `fig07.png` in this directory — it shows the Font page.
- **Font Effects** – set Font Color, Text Decoration, and Effects. Read the screenshot `fig08.png` in this directory — it shows the Font Effects page.
- **Borders** – configure Line Arrangement, Line style, and Padding.
- **Background** – apply None, Color, Gradient, Hatching, Bitmap, or Pattern fill. Read the screenshot `fig09.png` in this directory — it shows the Background page.
- **Shadow** – add a shadow; enable **Use shadow** first.

## Position and Size
Tables are treated as graphic objects. Use the **Position and Size dialog** to reposition or resize a table.
