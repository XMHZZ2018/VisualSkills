# Move / Reposition a Table (LibreOffice Impress 7.3.7)

Tables are placed into text boxes when created and are treated like any other graphic object in Impress. Only the Position and Size dialog can be used for precise positioning of tables.

## Opening the Position and Size Dialog

After selecting the table, use one of these methods:

- Right-click on the table and select **Position and Size** from the context menu.
- Go to **Format > Object and Shape > Position and Size** on the Menu bar.
- Press **F4**.

## Moving by Dragging

The position of the table text box can also be changed by clicking on the border and dragging to move the table text box to a new position.

> **Note:** When the size of a text box containing a table is changed, the table also increases or decreases in size to keep the text box filled with the table.

## Resizing

The table text box size can be changed by clicking and dragging on the selection handles.
