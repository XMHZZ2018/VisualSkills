# Insert and Resize Images

1. Insert an image via Insert > Image > From File, navigate to your file, and click Open.
2. Double-click the image on the slide to open the Position and Size dialog (or press F4).
3. In the Position and Size dialog, locate the Size section with Width and Height fields. Check 'Keep Ratio' to maintain the aspect ratio when resizing.

   Read the screenshot `step03.png` in this directory — it shows step 3.

4. Enter a new value in the Width field; with Keep Ratio checked, the Height will automatically adjust to preserve the aspect ratio. Click OK to apply.
5. To set width and height independently, uncheck 'Keep Ratio', then enter custom values in both Width and Height fields. Click OK.

   Read the screenshot `step05.png` in this directory — it shows step 5.

6. To resize freehand, click the image once to select it, then drag a corner handle to resize while keeping the aspect ratio, or drag a side/top/bottom handle to stretch in one dimension only.
