# Add and Edit Text Boxes

1. Go to Insert > Text Box in the menu bar.

   Read the screenshot `step01.png` in this directory — it shows step 1.

2. Click and drag on the slide to draw the text box at the desired size and position.
3. Type your content directly into the text box.
4. To edit existing text, double-click the text box to enter edit mode, then click to position the cursor or select text to modify.
5. To move a text box, click once on its border (so the box is selected but not in edit mode) and drag it to a new position.

   Read the screenshot `step05.png` in this directory — it shows step 5.

6. Click outside the text box to deselect it and return to the normal slide view.
