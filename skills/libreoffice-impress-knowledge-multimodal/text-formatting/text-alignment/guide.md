# Text Alignment (LibreOffice Impress 7.3.7)

Text alignment (left, center, right) is a paragraph formatting option applied to text in text boxes. It can be set using the Text Formatting toolbar, the Paragraph dialog, or the Sidebar Paragraph panel.

## Selecting Text First

Before applying alignment, select the text to format:

- To align all text in a text box, click once on the **border** of the text box to display selection handles. Formatting changes apply to all text in the box.
- To align only part of the text:
  - Click and drag the cursor over the text to highlight it.
  - Double-click to select a word; triple-click to select a whole paragraph.
  - Click in the text, then hold **Shift** and use the arrow keys to select text.

## Method 1: Text Formatting Toolbar

Use the alignment tools directly on the **Text Formatting toolbar**. Formatting is applied immediately.

## Method 2: Paragraph Dialog

1. Select the text or paragraph to align.
2. Open the Paragraph dialog using one of:
   - Right-click the selected text and choose **Paragraph** from the context menu.
   - Go to **Format > Paragraph** on the Menu bar.
3. Read the screenshot `fig04.png` in this directory — it shows the Paragraph dialog with the Indents & Spacing page open.
4. Use the alignment options available in the dialog tabs.
5. Click **OK** to save changes and close the dialog.
6. Click outside the text box to deselect.

## Method 3: Sidebar Paragraph Panel

1. Select the text box so that selection handles and border are displayed.
2. On the Sidebar, click **Properties** to open the Properties deck.
3. Click **Paragraph** to open the Paragraph panel.
4. Read the screenshot `fig05.png` in this directory — it shows the Paragraph panel in the Properties deck on the Sidebar.
5. Use the alignment tools in the Paragraph panel. Formatting is applied immediately.
6. If needed, click **More Options** on the right of the title bar to open the full Paragraph dialog.
