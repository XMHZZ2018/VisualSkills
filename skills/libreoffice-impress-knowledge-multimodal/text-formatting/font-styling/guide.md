# Font Styling

1. Open LibreOffice Impress and click on a text box to place your cursor inside it.
2. Select the text you want to format by clicking and dragging over it with your mouse.
3. Use the formatting toolbar at the top to change the font type (dropdown showing font name), size (number field), and apply Bold (Ctrl+B), Italic (Ctrl+I), or Underline (Ctrl+U).

   ![Step 3](step03.png)

4. To change font color, click the small arrow next to the Font Color button (the 'A' icon with a colored bar beneath it) in the toolbar to open the color palette, then click your desired color.

   ![Step 4](step04.png)

5. For more color options, go to Format > Character > Font Effects tab, click the Font Color dropdown, and choose a color or select Custom Color for a full color picker. Click OK to apply.

   ![Step 5](step05.png)
