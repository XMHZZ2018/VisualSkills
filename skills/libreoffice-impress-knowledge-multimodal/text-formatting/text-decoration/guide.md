# Bold, Underline, and Strikethrough (LibreOffice Impress 7.3.7)

Bold, underline, and strikethrough are character formatting options applied as direct or manual formatting. Direct character formatting overrides any formatting applied through styles.

## Selecting Text

Text must be selected before formatting is applied. Use one of the following:

- Click and drag the cursor over the text to highlight it.
- Double-click to select a word; triple-click to select a whole paragraph.
- Click in the text, then hold **Shift** and use the arrow keys to extend the selection.
- To format all text in a text box, click once on the border of the text box to display the selection handles.

## Applying Character Formatting

1. Select the characters or words to format.
2. Apply formatting using one of the following methods:
   - **Text Formatting toolbar** — click the Bold, Underline, or Strikethrough buttons directly. Formatting is immediate.
   - **Format menu** — go to **Format** on the Menu bar and select a formatting option. This opens a context menu or dialog with further options.
   - **Character dialog** — see below.
   - **Sidebar** — use the Character panel in the Properties deck. Formatting is immediate.
3. Click outside the text box to deselect the text.

## Character Dialog

Read the screenshot `fig06.png` in this directory — it shows the Character dialog Fonts page (Figure 70), where font effects including bold, underline, and strikethrough are configured.

1. Open the Character dialog using one of the following methods:
   - Right-click on the selected text and select **Character** from the context menu.
   - Go to **Format > Character** on the Menu bar.
2. Select the characters or words in the text box for formatting.
3. Use the options on the tabbed pages of the Character dialog to apply the desired formatting.
4. Click **OK** to save the changes and close the dialog.
5. Click outside the text box to deselect it.
