# Change Font Family / Typeface (LibreOffice Impress 7.3.7)

## Selecting Text First

Before applying font formatting, select the text using one of these methods:

- Click the border of a text box (selection handles appear) — changes apply to all text in the box.
- Click and drag over text to highlight it.
- Double-click to select a word; triple-click to select a paragraph.
- Click in text, hold Shift, and use arrow keys to extend selection.

## Changing the Font (Character Formatting)

Use any of these methods to change the font family of selected characters or words:

**Method 1 — Text Formatting toolbar**
Use the font name drop-down on the Text Formatting toolbar. Changes apply immediately.

**Method 2 — Character dialog**
1. Select the characters or words to format.
2. Open the Character dialog using one of:
   - Right-click the selected text and choose **Character** from the context menu.
   - Go to **Format > Character** on the Menu bar.
3. On the **Fonts** page, choose the desired font family.
   Read the screenshot `fig06.png` in this directory — it shows the Character dialog Fonts page where font family is set.
4. Click **OK** to apply and close the dialog.
5. Click outside the text box to deselect.

**Method 3 — Sidebar Character panel**
Use the Character panel in the **Properties** deck on the Sidebar. Formatting is immediate.

## Notes on Styles vs. Direct Formatting

- Text in AutoLayout boxes uses **Presentation Styles**; text in plain text boxes uses **Drawing Styles** (Default Drawing Style by default).
- Direct formatting overrides any style-based formatting.
- For consistent formatting across many slides, modifying the relevant style is recommended over manual formatting.
