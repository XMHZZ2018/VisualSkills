# Change Font Size (LibreOffice Impress 7.3.7)

## Quick Font Resizing

After selecting text, the font size can be quickly increased or decreased using:

- **Increase Font Size**: Ctrl+] (macOS ⌘+])
- **Decrease Font Size**: Ctrl+[ (macOS ⌘+[)

These tools are on the Text Formatting toolbar. The amount of change depends on the standard sizes available for the font in use.

> **Note:** AutoLayout text boxes automatically adjust font size of text inserted into them. When a long piece of text is inserted, the font size may decrease to fit the box.

## Selecting Text First

Text must be selected before formatting can be applied:

- To format **all text** in a text box, click once on the border to display selection handles.
- To format **part of the text**, use one of:
  - Click and drag the cursor over the text.
  - Double-click to select a word; triple-click to select a whole paragraph.
  - Click, then hold Shift and use arrow keys.

## Changing Font Size via Text Formatting Toolbar

1. Select the text to format.
2. Use the font size field or Increase/Decrease Font Size buttons on the Text Formatting toolbar.

## Changing Font Size via Character Dialog

1. Select the characters or words to format.
2. Open the Character dialog using one of:
   - Right-click the selected text and select **Character** from the context menu.
   - Go to **Format > Character** on the Menu bar.
3. Read the screenshot `fig06.png` in this directory — it shows the Character dialog Fonts page where font size can be set.
4. Use the options on the tabbed pages to set the desired font size.
5. Click **OK** to save and close the dialog.
6. Click outside the text box to deselect.

## Changing Font Size via Sidebar Character Panel

1. Select the text box so that selection handles and border are displayed.
2. On the Sidebar, click **Properties** to open the Properties deck.
3. Click **Character** to open the Character panel.
4. Set the font size using the available tools. Formatting is applied immediately.
5. If needed, click **More Options** to open the full Character dialog.
