# Change Font Color (LibreOffice Impress 7.3.7)

## Select the Text

Before changing font color, text must be selected using one of these methods:

- Click once on the **border of the text box** (selection handles appear) to apply formatting to all text in the box.
- Click and drag the cursor over text to highlight a portion.
- Double-click to select a word; triple-click to select a whole paragraph.
- Click in the text, then hold **Shift** and use the arrow keys to extend the selection.

## Change Font Color

Apply font color using any of the following methods:

### Method 1: Text Formatting Toolbar
Use the font color tool directly on the **Text Formatting toolbar**. Formatting is applied immediately.

### Method 2: Format Menu
Go to **Format** on the Menu bar and select a character formatting option from the drop-down menu.

### Method 3: Character Dialog

1. Open the **Character dialog** using one of:
   - Right-click on the selected text and select **Character** from the context menu.
   - Go to **Format > Character** on the Menu bar.
2. Select the characters or words in the text box for formatting.
3. Use the options on the tabbed pages of the Character dialog to set the font color.
   Read the screenshot `fig06.png` in this directory — it shows the Character dialog Fonts page.
4. Click **OK** to save the changes and close the dialog.
5. Click outside the text box to deselect it.

### Method 4: Sidebar Character Panel
Use the options in the **Character panel** in the **Properties deck** on the Sidebar. Formatting is applied immediately.

## Notes

- Direct character formatting overrides any formatting applied via presentation or drawing styles.
- Click outside the text box to deselect the text when done.
