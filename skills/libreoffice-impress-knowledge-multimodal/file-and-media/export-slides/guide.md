# Export Slides as Images (LibreOffice Impress 7.3.7)

## Export in Other Formats (e.g., PNG)

Use this method to export slides as image files such as PNG, GIF, or JPG.

1. Make sure the presentation file is open in Impress.
2. Go to **File > Export** on the menu bar to open a file browser window and navigate to where the exported file is to be saved.
3. Enter a name for the exported file in the **Name** text box at the top of the file browser window.
4. At the bottom of the file browser window, select the required format (e.g., PNG, GIF, JPG) from the drop-down list.
5. Click on **Save** to export the file to the selected location. Depending on the format selected, an options dialog may open allowing further options to be selected.
6. Click on **OK** to save the exported file and close the options dialog.

## Web Pages (HTML) Export — Saving Slides as Images

When exporting as HTML, you can choose the image format for slides (PNG, GIF, or JPG).

1. Make sure the presentation file is open in Impress.
2. Create a folder to contain the HTML pages that are generated and select it as the destination.
3. Go to **File > Export** on the menu bar to open a file browser window and navigate to the save location.
4. Enter a name for the HTML file in the **Name** text box.
5. At the bottom of the file browser window, select **HTML Document (Impress) (.html; .htm)** from the drop-down list.
6. Click on **Save** — the HTML Export dialog opens at the Assign Design page. Read the screenshot `fig02.png` in this directory — it shows the HTML Export dialog Assign Design page.
7. Select a design for the HTML pages (existing or new), then click **Next**.
8. Select the publication type. Read the screenshot `fig03.png` in this directory — it shows the HTML Export dialog Publication Type page with options including Standard HTML, Standard HTML with frames, Single-document HTML, Automatic, and WebCast. Click **Next**.
9. Select the format for saving images (**PNG**, **GIF**, or **JPG**) and the monitor resolution. Read the screenshot `fig04.png` in this directory — it shows the HTML Export dialog Save Images As page with image format and monitor resolution options. Click **Next**.
10. Enter title page details (author, email, home page) if applicable, then click **Next**.
11. Select the style of navigation buttons. Read the screenshot `fig05.png` in this directory — it shows the HTML Export dialog Select Button Style page. Click **Next**.
12. Select the color scheme for the web pages, then click **Create** to generate the HTML files.

> **Note:** Exporting as web pages (HTML) does not retain any animation effects or slide transitions.
