# Save / Save As (LibreOffice Impress 7.3.7)

## Saving a New Slide Show

Slide shows are saved in Open Document Presentation (ODP) format by default, but Impress can save in other formats.

1. Create a new slide show.
2. Open the file browser using one of these methods:
   - Click **Save** on the Standard toolbar
   - Go to **File > Save** on the Menu bar
   - Use **Ctrl+S** (macOS **⌘+S**)
3. Navigate to the folder where the slide show is to be saved.
4. Enter a filename in the **File name** / **Name** / **Save as** text box.
5. Select a file format from the drop-down list at the bottom of the file browser. Default is **ODF Presentation (.odp)**.
6. Click **Save**.

## Saving in ODP Format

1. Open a slide show and make all required changes.
2. Save using one of these methods:
   - Click **Save** on the Standard toolbar
   - Go to **File > Save** on the Menu bar
   - Use **Ctrl+S** (macOS **⌘+S**)

## Saving in Other Presentation Formats (Save As)

1. Open a slide show and make all required changes.
2. Open the Save As dialog using one of these methods:
   - Click the triangle **▼** to the right of **Save** on the Standard toolbar and select **Save As**
   - Go to **File > Save As** on the Menu bar
   - Use **Ctrl+S** (macOS **⌘+Shift+S**)
3. In the file browser, navigate to the folder where the slide show is to be saved.
4. Enter a filename in the **Name** text box.
5. Select a presentation file format from the drop-down list at the bottom of the file browser. Default is **ODF Presentation (.odp)**.
6. Click **Save**.
