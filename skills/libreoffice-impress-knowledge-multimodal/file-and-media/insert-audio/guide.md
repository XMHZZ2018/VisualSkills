# Insert Audio (LibreOffice Impress 7.3.7)

## Inserting Audio or Video

### Using a File Browser

1. Go to **Insert > Audio or Video** on the Menu bar. A file browser opens — read the screenshot `fig01.png` in this directory — it shows Figure 230: the Insert Audio or Video file browser.
2. Navigate to the folder where the audio or video file is located. Only compatible audio and video files will be available.
3. Select a compatible file and click **Open** to embed it in the center of the slide. The Media Playback toolbar opens automatically.
4. Reposition and/or resize the audio or video file as needed.

### Using the Gallery

1. Open the Gallery deck using one of these methods:
   - Go to **View > Sidebar** on the Menu bar, then select **Gallery**.
   - Press **Ctrl+F5** to open the Sidebar, then select **Gallery**.
   - Go to **View > Gallery** on the Menu bar.
2. Browse to a theme containing media files, for example **Sounds**.
3. Click the audio or video option required and drag it onto the slide. The Media Playback toolbar opens automatically.
4. Reposition and/or resize the audio or video file as needed.

## Media Playback

The Media Playback toolbar opens automatically when an audio or video file is selected. Read the screenshot `fig02.png` in this directory — it shows Figure 231: the Media Playback toolbar.

The toolbar contains the following tools (left to right):

- **Insert Audio or Video** — opens the Insert Audio or Video file browser.
- **Play** — plays the media.
- **Pause** — pauses playback.
- **Stop** — stops playback.
- **Repeat** — repeats playback until deselected.
- **Position** — selects the position within the media file to start playing from.
- **Mute** — suppresses the volume.
- **Volume** — adjusts the volume.
- **Media Path** — indicates where the media file is stored on the computer.
