# Apply Slide Layouts

1. Open the Layouts panel by clicking View > Sidebar and selecting the Layouts section, or use the View menu to access the Layouts panel directly.
2. In the slide panel on the left, click to select the slide you want to change the layout for.
3. Go to View > Sidebar (or press Ctrl+F5) and click the Layouts icon in the sidebar to open the Layouts panel showing all predefined layouts (Title Slide, Title Content, Blank, etc.).

   Read the screenshot `step03.png` in this directory — it shows step 3.

4. Alternatively, right-click on the slide in the main editing area and choose Slide Layout from the context menu to see available layouts.

   Read the screenshot `step04.png` in this directory — it shows step 4.

5. Click on the desired layout thumbnail (e.g., Title, Content, Blank, Title Only) to apply it to the selected slide immediately.
6. If placeholder positions shifted after changing the layout, drag the content or title placeholders to reposition them as needed on the slide.
