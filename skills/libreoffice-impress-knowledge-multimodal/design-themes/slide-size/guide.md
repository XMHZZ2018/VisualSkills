# Change Slide Size and Orientation

1. Open the Format menu from the menu bar and select 'Page...' (or 'Slide Size...').
2. In the Slide Size dialog, set the Width to 16.00" and the Height to 9.00" to use a widescreen 16:9 format. For a standard 4:3 format, use 10.00" × 7.50". To switch orientation, swap the width and height values (e.g., portrait: 7.50" × 10.00").

   Read the screenshot `step02.png` in this directory — it shows step 2.

3. Click OK to apply the new slide dimensions. If prompted, choose whether to scale existing content or keep it as-is.
4. To set a background image, go to Format > Page... again, then click the 'Background' tab and choose 'Bitmap' from the Area dropdown.

   Read the screenshot `step04.png` in this directory — it shows step 4.

5. Select your desired bitmap from the list, then under Style choose 'Stretched' (or 'AutoFit') so the image fills the entire slide without tiling.
6. Click OK and confirm to apply the background to all slides.
