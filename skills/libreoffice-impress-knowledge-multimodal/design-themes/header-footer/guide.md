# Add Header and Footer

1. Open your document. Move your mouse cursor to the very top of the page, above the top margin boundary (you will see two small corner markers indicating where the margin begins).

   Read the screenshot `step01.png` in this directory — it shows step 1.

2. Click in that area above the margin line. A grey bar labelled 'Header (Default Page Style)' will appear with a '+' button. Click the '+' to insert the header.

   Read the screenshot `step02.png` in this directory — it shows step 2.

3. Your cursor is now inside the header area. Use the alignment buttons in the toolbar (e.g., Align Right) to position your header text where you want it.
4. Type any static text you want to appear on every page (such as your name or document title).
5. To insert a dynamic page number (one that updates automatically per page), go to Insert > Header and Footer > Field > Page Number. This inserts a field rather than a static number.

   Read the screenshot `step05.png` in this directory — it shows step 5.

6. To add a footer, scroll to the bottom of the page and repeat the same process: click below the bottom margin boundary to reveal the 'Footer (Default Page Style)' bar, then click '+' to activate it.
