# Set Slide Background (LibreOffice Impress 7.x)

## For a single slide

1. Select the slide you want to change in the slide panel on the left.
2. Go to **Slide** menu (in the menu bar) > **Slide Properties...**.

   Read the screenshot `step02.png` in this directory — it shows slide menu showing slide properties.

3. In the Slide Properties dialog, click the **Background** tab. Select **Color** as the background type (it's the default button).

   Read the screenshot `step03.png` in this directory — it shows background tab with color selected.

4. Pick a color from the palette, or use the **Pick** button to enter a custom hex value. The "Active" swatch shows the current color; "New" shows your selection.

   Read the screenshot `step05.png` in this directory — it shows color selected with preview.

5. Click **OK**. The background is applied to that single slide only.

## For all slides

**Important:** In LibreOffice Impress 7.x, there is NO "apply to all slides" prompt. Slide Properties only applies to the currently selected slide. To set the same background on all slides, you must repeat steps 1-5 for each slide individually:

1. Click slide 1 in the slide panel.
2. Slide > Slide Properties > Background > set the color > OK.
3. Click slide 2, repeat.
4. Continue for all remaining slides.

**Tip:** After setting the color on the first slide, subsequent slides will show the color in the "Recent Colors" row at the bottom of the palette, making it faster to select.
