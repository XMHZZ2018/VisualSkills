# Creating lists (LibreOffice Impress 7.3)

## AutoLayout text boxes

Default lists are unordered. To create a list:

1. Go to **View > Normal**
2. Select a slide layout (right-click slide, **Slide > Layout**, or Sidebar **Layouts** panel)
3. Click **"Click to add Text"** and type the first item
4. Press **Enter** to create a new list point; **Shift+Enter** for a new line without a new point

**Demote** (increase outline level): **Tab**, **Alt+Shift+Right Arrow** (macOS: **⌥+Shift+Right Arrow**), **Demote** on Outline toolbar, Sidebar Lists panel, or **Format > Lists > Demote**

Read the screenshot `figure75.png` in this directory — it shows the Lists panel in the Properties deck on the Sidebar

**Promote** (decrease outline level): **Shift+Tab**, **Alt+Shift+Left Arrow** (macOS: **⌥+Shift+Left Arrow**), **Promote** on Outline toolbar, Sidebar Lists panel, or **Format > Lists > Promote**

**Move Down** (same level): **Alt+Shift+Down Arrow** (macOS: **⌥+Shift+Down Arrow**), **Move Down** on Outline toolbar, Sidebar Lists panel, or **Format > Lists > Move Down**

**Move Up** (same level): **Alt+Shift+Up Arrow** (macOS: **⌥+Shift+Up Arrow**), **Move Up** on Outline toolbar, Sidebar Lists panel, or **Format > Lists > Move Up**

Read the screenshot `figure73.png` in this directory — it shows the Outline toolbar

> Note: Do not change outline style by selecting a list point and choosing another outline style — this does not work in Impress.

## Text boxes

1. Create a text box with cursor active
2. Enable list using one of:
   - **Toggle Unordered List** or **Toggle Ordered List** on the Text Formatting toolbar
   - Same buttons in Sidebar **Lists** panel
   - **Format > Lists > Unordered List** or **Ordered List**
3. Press **Enter** for new list point; **Shift+Enter** for new line without new point

**Increase indent**: **Tab**, **Alt+Shift+Right Arrow**, **Demote** on Outline toolbar/Sidebar, or **Format > Lists > Demote**

**Decrease indent**: **Shift+Tab**, **Alt+Shift+Left Arrow**, **Promote** on Outline toolbar/Sidebar, or **Format > Lists > Promote**

**Move Down / Move Up**: same shortcuts and toolbar/menu options as AutoLayout above

## Formatting lists

- **Bullets and Numbering dialog**: change list type, properties, size, position, scope (entire list or single point)
- **Sidebar Lists panel**: change list type, promote/demote, move up/down

Read the screenshot `figure71.png` in this directory — it shows the Character panel in the Properties deck on the Sidebar

- **Outline toolbar**: promote/demote, move up/down
- **Text Formatting toolbar**: change list type; click **▼** next to **Toggle Unordered/Ordered List** then **More Bullets** or **More Numbering** to open advanced dialogs

Read the screenshot `figure77.png` in this directory — it shows more numbering types available in the advanced numbering dialog
