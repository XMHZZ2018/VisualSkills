# Creating lines and shapes (LibreOffice Impress 7.3)

## Drawing toolbar shape categories

The Drawing toolbar provides access to these regular shape categories (click the triangle next to the icon to open each sub-toolbar):

| Category | Description |
|---|---|
| Rectangle | Basic rectangle and square shapes |
| Ellipse | Circles and ellipses |
| Basic Shapes | Rounded rectangles, parallelograms, trapezoids, circles, arcs, rings, frames, crosses, etc. |
| Symbol Shapes | Smiley faces, suns, moons, brackets, braces, etc. |
| Block Arrows | Directional arrows, chevrons, pentagons, arrow callouts |
| Flowchart | Process, decision, document, data, and other flowchart shapes |
| Callout Shapes | Callout boxes with pointers |
| Stars and Banners | Multi-point stars, scrolls, doorplates |
| 3D-Objects | Cubes, spheres, cones, cylinders, etc. |

## Drawing regular shapes

1. Select the required tool from the Drawing toolbar or from a sub-toolbar (click the triangle next to the icon).
2. Place the cursor on the slide, then click and drag to define an enclosing rectangle.
3. Release the cursor and the shape is drawn.

**Modifier keys while drawing:**
- Hold **Shift** to maintain equal height and width (e.g. draw a square or circle).
- Hold **Alt** to draw the shape from its center.
- Hold **Shift + Alt** to draw an equal-sided shape from the center.
- Hold **Ctrl** to snap the top-left corner to the nearest grid point.

## Regular shape adjustment

After creating a shape, colored dots (different from selection handles) may appear. These dots adjust shape-specific properties:

**Basic Shapes:** Rectangle Rounded (corner radius), Parallelogram/Trapezoid (internal angles), Circle Pie (sector size), Isosceles Triangle (shape/type), Hexagon/Octagon (angles), Cylinder/Cube (perspective), Frame (thickness), Cross (arm thickness), Ring (internal diameter).

**Block Arrows:** Right/Left/Up/Down Arrow (shape and thickness), Chevron/Pentagon (angle), Arrow Callouts (shape and thickness).

**Flowchart:** Standard flowchart shapes — process, decision, document, data, etc.

**Stars and Banners:** 4/8/24-Point Star (thickness and shape), Vertical/Horizontal Scroll (width and shape), Doorplate (corner curvature).

## Drawing lines and arrows

**Straight lines:**
1. Click **Insert Line** on the Drawing toolbar, or click the triangle next to **Lines and Arrows** and select **Insert Line**.
2. Click at the start position and drag the cursor.
3. Release the cursor at the end position.

**Modifier keys for lines:**
- **Ctrl** — snap end of line to nearest grid point.
- **Shift** — restrict drawing angle to multiples of 45 degrees.
- **Alt** — extend line symmetrically from the center (start point becomes midpoint).

**Arrows:**
1. Click the triangle next to **Lines and Arrows** on the Drawing toolbar and select the arrow type.
2. Click at the starting position, then drag. Arrowhead(s) are drawn at the end on release.
3. Same modifier keys apply (Ctrl, Shift, Alt).

## Drawing curves, polygons and freeform lines

**Curves:**
1. Click triangle next to **Curves and Polygons** on the Drawing toolbar.
2. Select **Curve** or **Curve Filled**.
3. Click and drag to create the starting point.
4. Release and drag to draw a curve, bending the line.
5. Click to set the end point and fix the curve.
6. Continue drawing straight lines from each click point.
7. Double-click to end. If **Curve Filled** was selected, the shape is automatically filled.

**Polygons:**
1. Open **Curves and Polygons** sub-toolbar.
2. Select **Polygon**, **Polygon Filled**, **Polygon (45°)** or **Polygon Filled (45°)**.
3. Click and drag to draw the first line from the start point.
4. Move and click for each corner point.
5. Double-click to end. Filled variants auto-fill with default color.

**Freeform lines:**
1. Select **Freeform Line** or **Freeform Line Filled**.
2. Click and drag to draw the shape.
3. Release to complete. Filled variant auto-fills between start and end points.

**Note:** Hold **Shift** when drawing curves or polygons to restrict angles to 45 or 90 degrees.

## Grouping objects

Grouping puts objects into a container so they behave as one object. Changes apply globally to all objects in the group.

**Temporary grouping (multi-select):**
- Hold **Shift** and click multiple objects.
- Or click and drag a marquee around multiple objects.
- Click outside to cancel the temporary group.

**Permanent grouping:**
- Select objects, then right-click > **Group** or **Format > Group > Group**.
- To ungroup: right-click > **Ungroup** or **Format > Group > Ungroup**.
