# Working with slides (LibreOffice Impress 7.3)

## Inserting new slides
New slide is inserted after the selected slide, inheriting its slide master.

- **Slide > New Slide**
- Right-click slide in Slides pane → **New Slide**
- Keyboard: `Ctrl+M` (macOS: `⌃+M`)

To insert at end: deselect all slides in Slides pane, then use any method above.

## Inserting slides from a file
1. Select target slide (or deselect all to append at end)
2. **Slide > Insert Slide from File** → navigate to file → **Open**
Read the screenshot `figure232.png` in this directory — it shows the Insert File dialog
3. Click triangle next to filename to expand slide list
4. Select slides; optionally enable **Link** (embeds as OLE) or **Delete unused backgrounds**
5. Click **OK**
Read the screenshot `figure233.png` in this directory — it shows the Insert Slides/Objects dialog

## Copying and pasting between presentations
1. Open source presentation in **Slide Sorter** view; select and copy slides (`Ctrl+C` / macOS `⌘C`)
2. Open target presentation in **Slide Sorter** view; select insertion point (or deselect all to append)
3. Paste: `Ctrl+V` / macOS `⌘V`

## Dragging and dropping between presentations
1. Open both presentations in **Slide Sorter** view
2. Select target insertion slide in target presentation
3. From source, drag slides to target — hold `Ctrl` (macOS `⌘`) while dragging to **copy** instead of move

## Duplicating slides
Preserves formatting, layout, and animations.

1. Open **Normal** or **Slide Sorter** view; select slides
2. **Slide > Duplicate Slide** (or right-click → **Duplicate Slide**)

Duplicated slides are inserted after the selected slides.

## Renaming slides
1. Select slide in **Normal** or **Slide Sorter** view
2. **Slide > Rename Slide** (or right-click → **Rename Slide**)
3. Type new name → **OK**

## Expanding slides
Splits a slide's content into multiple slides using first outline level as new titles.

1. Select slide in Slides pane
2. **Slide > Expand Slide**
Read the screenshot `figure235.png` in this directory — it shows an example of expanding slides

## Creating summary slides
Creates an agenda slide at the end using slide titles as bullet points.

1. Select the first slide to appear in the summary
2. **Slide > Summary Slide**
