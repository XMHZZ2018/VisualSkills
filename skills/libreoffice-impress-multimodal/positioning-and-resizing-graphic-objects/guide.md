# Positioning and resizing graphic objects (LibreOffice Impress 7.3)

## Using the cursor

**Positioning**
1. Click object to display selection handles.
2. Hover over object until cursor changes shape.
3. Click and drag to desired position (ghost image assists).
4. Release when positioned. Arrow keys also move selected objects.

**Resizing**
1. Click object to display selection handles.
2. Hover over a selection handle.
3. Click and drag handle to resize (ghost image assists).
4. Hold **Shift** before dragging to maintain width/height ratio; release cursor before releasing Shift.

## Position and Size dialog (precise control)

Open with any of:
- Keyboard shortcut **F4**
- **Format > Object and Shape > Position and Size**
- Right-click > **Position and Size**
- **More Options** in the Position and Size panel on the Sidebar

**Positioning**
1. Open dialog, click **Position and Size** tab.
2. Set **Position X** (horizontal) and **Position Y** (vertical) — measured from selected Base point (default: top-left of slide).
3. To lock position, select **Position** in the Protect section.
4. Click **OK**.

**Resizing**
1. Open dialog, click **Position and Size** tab.
2. Select a **Base point** anchor (default: top-left corner stays fixed).
3. To maintain proportions, enable **Keep ratio** before changing dimensions.
4. Set **Width** and **Height**.
5. To lock size, select **Size** in the Protect section.
6. Click **OK**.

## Sidebar Position and Size panel

1. Select object; click **Properties** in Sidebar to open Properties deck.
2. Click **Position and Size** to open the panel.
3. Set **Position X** and **Position Y** (relative to Base point set in the dialog).
4. Press **Enter** to apply.
