# Inserting images (LibreOffice Impress 7.3)

## Inserting from a file

1. Insert a new slide or select an existing slide.
2. If necessary, select a layout for the slide.
3. Go to **Insert > Image** on the Menu bar — the Insert Image file browser opens.
Read the screenshot `figure98.png` in this directory — it shows an example of the Insert Image file browser
4. Navigate to the directory containing the image and select the file.
5. If necessary, select **Preview** to see a thumbnail in the preview pane.
6. Click **Open** to place the image in the center of the slide. Selection handles appear around the image border.
7. Format the image as needed (see "Formatting images").

## Linking instead of embedding

In the Insert Image file browser, select **Insert as Link** to link rather than embed the image.

Use linking when:
- The image file is large and you want to reduce presentation file size.
- The same image is reused across many presentations (e.g., a shared background).
- The linked file is only available when the presentation is opened.

> Prefer embedding when the presentation will be copied and used on other computers.

## Inserting a scanned image

Requirements: scanner driver installed, and one of:
- Linux/UNIX: scanner supported by SANE
- Windows/macOS: scanner is TWAIN compatible

**Steps:**

1. Prepare the image and ensure the scanner is switched on.
2. First use: go to **Insert > Media > Scan > Select Source** to open a scanner dialog.
3. Subsequent use: go to **Insert > Media > Scan > Request** to open a scanner dialog.
4. If multiple scanners are connected, select the scanner from the **Select Source** or **Device Used** drop-down list.
5. If available, click **Create Preview** or **Preview** to preview the scan.
6. Crop and adjust the scan as needed.
7. Click **Scan** — Impress places the scanned image in the center of the selected slide.
8. Format the image as needed (see "Formatting images").
