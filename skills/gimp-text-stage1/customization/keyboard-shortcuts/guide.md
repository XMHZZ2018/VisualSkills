# Creating Keyboard Shortcuts (GIMP 2.10)

GIMP gives you two ways to assign keyboard shortcuts: a quick dynamic method right in the menus, or a dedicated dialog for full control.

**The quick way — dynamic shortcuts.** Head to **Edit → Preferences**, open the **Interface** section, and enable **Use dynamic keyboard shortcuts**. While you're there, check **Save keyboard shortcuts on exit** so nothing gets lost. Now just hover over any menu command, keep your mouse still, and press your desired key combo — it appears right next to the command. Stick with **Ctrl+Alt+Key** combinations to avoid clashing with built-in shortcuts.

**The thorough way — the Keyboard Shortcuts dialog.** Open it via **Edit → Keyboard Shortcuts…** (or from Preferences, click **Configure Keyboard Shortcuts…**). This dialog lets you assign shortcuts to menu commands, tools, and filters — anything GIMP exposes as an action.

Browse the action categories by expanding them with **+**, or type in the **Search** field to filter the list. Click the action you want to rebind — the Shortcut column changes to "New accelerator…" — then press your key combination. If it's already taken, GIMP asks whether to reassign it or cancel.

To remove an existing shortcut, select the action and press **Backspace**.

The **Save** button writes changes to disk immediately. If you just close the dialog, your shortcuts stay active for the session but only persist if **Save keyboard shortcuts on exit** is enabled at the bottom of the dialog.
