# Symmetry Painting (GIMP 2.10)

Symmetry Painting lets you paint with automatic mirroring, tiling, or radial repetition using any brush-based tool — Pencil, Paintbrush, Eraser, Airbrush, MyPaint brush, Clone, Smudge, Dodge, and Ink.

Open the dialog via **Windows > Dockable Dialogs > Symmetry Painting**. You'll see a single **Symmetry** dropdown set to **None** by default. Pick a mode and green dotted guide lines appear on canvas — just start painting.

**Mirror** reflects your strokes across an axis. Check **Horizontal Symmetry**, **Vertical Symmetry**, or **Central Symmetry** (or combine them). The axis defaults to the image center, but you can reposition it with **Vertical axis position** and **Horizontal axis position**. Enable **Disable brush transform** if you want the stroke mirrored but the brush tip itself kept unflipped — mostly visible with asymmetric brushes.

**Tiling** repeats your stroke in a grid across the canvas — great for seamless patterns. Set **Interval X** and **Interval Y** (pixel spacing between copies), **Shift** (horizontal offset between rows), and optionally **Max strokes X/Y** to cap repetitions (0 means unlimited).

**Mandala** distributes strokes radially around a center point. Set the center with **Center abscissa** and **Center ordinate**, choose **Number of points** for how many copies appear around the ring, and toggle **Disable brush transform** as needed.

To turn symmetry off, just set the dropdown back to **None**.
