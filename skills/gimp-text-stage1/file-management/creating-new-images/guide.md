# Creating New Images (GIMP 2.10)

To create a blank canvas, go to **File > New…** or hit **Ctrl+N**. This opens the "Create a New Image" dialog where you set up everything before you start working.

At the top, you'll find a **Template** dropdown — a quick way to pick common presets (web banners, print sizes, etc.) without typing dimensions manually. If you have a go-to size you reuse often, you can save your own template later via **File > Create Template…**.

Below that, set your **Width** and **Height**. Default unit is pixels, but you can switch to inches, millimeters, or others from the adjacent menu. The **Portrait/Landscape** buttons simply swap width and height values for you.

Click the triangle at the bottom to expand **Advanced Options**. Here you control **X/Y resolution** (mainly matters for print — 300 ppi is standard), **Color space** (RGB for color work, Grayscale for B&W), and **Precision** (8-bit integer is fine for most work; bump to 16- or 32-bit for high-end editing with smoother gradients).

The **Fill with** setting determines your starting background — choose from Foreground color, Background color, White, Transparency (adds an alpha channel automatically), or the currently active Pattern.

You can also attach a **Comment** that travels with the file when saved as PNG, JPEG, or GIF.

Once everything looks right, hit **OK** and your new canvas opens ready to go. If you already have an image open, the new one appears in a separate window sized to match your first image by default.
