# Creating Bibliographies Manually (Zotero 7)

Sometimes you don't need the full word processor plugin — you just want to grab a few references and drop them somewhere. Zotero gives you two fast ways to do this: drag-and-drop (Quick Copy) and the right-click menu.

## Quick Copy (Drag-and-Drop)

The fastest route is Quick Copy. Select one or more items in Zotero's center column, then drag them straight into any text field — an email, a blog post editor, a plain document. Zotero instantly generates a formatted bibliography and drops it in. If you need an in-text citation instead of a full reference, just hold **Shift** as you start the drag.

Before you rely on this, set your default style. Go to **Zotero > Settings > Export** and pick the citation format you want Quick Copy to use. You can also configure site-specific styles and toggle HTML markup on or off from that same tab.

There's a clipboard variant too: select your items and hit **Edit > Copy Bibliography** (or **Ctrl/Cmd-Shift-C**). That copies the formatted references so you can paste them anywhere. For citations, use **Edit > Copy Citation** (or **Ctrl/Cmd-Shift-A**). Quick Copy also supports export formats like BibTeX and RIS, which is handy if you're feeding references into another tool.

## Right-Click to Create a Bibliography

For more control over the output, select items in Zotero, right-click, and choose **Create Bibliography from Item(s)…**. A dialog lets you pick a citation style and choose between a *Bibliography* or *Citations/Notes* list.

You then decide how to get the output: **Save as RTF** or **Save as HTML** for files, **Copy to Clipboard** for quick pasting, or **Print** to send it straight to a printer. The HTML option is especially nice for web publishing since it embeds metadata that other Zotero users can capture.
