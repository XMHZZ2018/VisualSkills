# Master Slides & Templates (LibreOffice Impress 7.3.7)

A master slide is basically a blueprint — it controls the default background, colors, fonts, and layout for every slide built from it. Change something on the master, and it ripples out to all slides using it. You can have more than one master slide in a single presentation.

To browse what's available, open the **Master Slides** deck on the Sidebar. It's organized into three sections: **Used in This Presentation**, **Recently Used**, and **Available for Use**. Right-click any master slide thumbnail and choose **Apply to All Slides** or **Apply to Selected Slides** to assign it.

If the built-in masters aren't enough, you can pull in more from other templates. Go to **Slide > Change Slide Master**, then click **Load** in the Available Master Slides dialog. Pick a category and template, hit **OK**, then select the newly loaded design. Check **Exchange background page** to apply it everywhere, or leave it unchecked to target only selected slides.

Visual reference: the Load Master Slide dialog with Categories and Templates lists and a slide preview.

To create your own master slide, switch to **View > Master Slide**. Click **New Master** on the Master View toolbar (or **Slide > New Master**), then rename it via **Rename Master**. Design your background, placeholders, and text styles, then click **Close Master View** or go to **View > Normal** when you're done.

You can edit any existing master the same way — enter Master Slide view, select the master in the Slides pane, then open **Slide > Slide Properties** to tweak the **Slide**, **Background**, and **Transparency** tabs. Remember: changes here affect every slide using that master.

Visual reference: the Slide Properties dialog showing the Slide tab with Paper Format, Orientation, Margins, and Layout Settings.

If you've manually formatted something on a regular slide and want to snap it back to the master's style, select the object and go to **Format > Clear Direct Formatting** (or right-click and choose **Clear Direct Formatting**).

For templates, go to **File > New > Templates** (or press **Ctrl+Shift+N**) to open the Select a Template dialog. Filter by **Presentations**, pick one, and a new presentation opens with that design baked in. To save your own work as a reusable template, use **File > Templates > Save As Template**, give it a name and category, and optionally check **Set as default template**. You can reset the default back to blank anytime via **File > Templates > Manage Templates**, then right-clicking your template and choosing **Reset Default**.

Visual reference: the Save As Template dialog with fields for template name, category selection, and a Set as default template checkbox.
