# Data Tables (LibreOffice Impress 7.3.7)

To drop a table onto a slide, go to **Insert > Table** on the Menu bar, set the number of columns and rows in the dialog, and click **OK**. Alternatively, click **Table** on the Standard toolbar and drag across the grid to pick your dimensions — or hit **More Options** to type exact numbers.

Visual reference: the Insert Table dialog with fields for Number of columns and Number of rows.

Once the table lands, the **Table** toolbar appears automatically with tools for borders, colors, merging, and more. If it doesn't show, enable it via **View > Toolbars > Table**.

For quick styling, open the **Table Design** panel in the Properties sidebar. Pick a predefined design, then toggle options like **Header row**, **Total row**, **Banded rows**, **First column**, **Last column**, or **Banded columns** to fine-tune the look.

Visual reference: the Table Design panel in the Properties sidebar showing predefined styles and toggle options for Header row, Banded rows, and Banded columns.

To adjust cell alignment, right-click selected cells, choose **Align**, and pick **Align Top**, **Center Vertically**, or **Align Bottom**. You can also reach this via **Format > Align Text**.

Need to change font color for specific cells — say, red for negative values? Select the cells, then right-click and open **Table Properties**, or go to **Format > Table > Properties**. On the **Font Effects** tab, set the **Font color** dropdown to your desired color. The **Font** tab lets you change family, style, and size per cell selection.

To merge cells, select them and click **Merge Cells** on the Table toolbar, or use **Format > Table > Merge Cells**. To split a cell, right-click it and choose **Split Cells**, then set how many pieces and whether to split horizontally or vertically.

Visual reference: the Split Cells dialog with a count spinner and Horizontally/Vertically direction options.

Add or remove rows and columns by right-clicking and selecting **Insert** (then **Row Above**, **Row Below**, **Column Before**, or **Column After**) or **Delete** (then **Delete Row**, **Delete Column**, or **Delete Table**). These are also under **Format > Table**.

For even spacing, click **Optimize** on the Table toolbar to access **Distribute Columns Evenly** and **Distribute Rows Equally**, or use **Optimal Column Width** / **Optimal Row Height** to auto-fit content.

Borders are configured on the **Borders** tab of the **Table Properties** dialog — choose a preset layout, set line style/color/width, and adjust cell padding. The **Background** tab lets you fill cells with a solid color, gradient, bitmap, pattern, or hatch. A **Shadow** tab is also available for adding depth to the entire table.

Tables behave like any other graphic object for positioning — use the **Position and Size** dialog (**F4**) to place and resize them precisely on the slide.
