# Configuring Keyboard Shortcuts (OpenToonz 1.7)

OpenToonz ships with a full set of predefined shortcuts for tools and menu commands, but you can remap any of them or add new ones. Shortcuts are per-user, so everyone on a shared machine can have their own layout.

To set a shortcut, open **File → Configure Shortcuts…**. Browse the categorized folders (Tools, Tool Modifiers, menus) or type in the **Search** field at the top to find what you need. Select the command, then press the key or combo you want to assign — it binds immediately. To clear one, select the command and hit **Remove**.

You can save your whole layout as a named preset with **Save As**, load someone else's with **Load**, or wipe everything with **Clear All Shortcuts**. The **Export Current Shortcuts** button dumps your config for backup or sharing.

A handy trick: press and *hold* any tool shortcut (like **B** for Brush) to temporarily switch to that tool — releasing the key snaps you back to whatever you were using before.

**Common tool shortcuts:** **A** Animate, **S** Selection, **B** Brush, **G** Geometric, **F** Fill, **E** Eraser, **T** Tape, **K** Style Picker, **C** Control Point Editor, **V** Skeleton. Hold **Space** to pan, **Shift+Space** to zoom, **Ctrl/Cmd+Space** to rotate.

**Brush size on the fly:** **U**/**I** decrease/increase the max size, **H**/**J** decrease/increase the min size.

**Playback:** **P** play, **L** loop, **,** / **.** step through drawings, **Shift+,** / **Shift+.** step through frames, **Ctrl/Cmd+,** / **Ctrl/Cmd+.** jump between keyframes.

**Rendering:** **Ctrl/Cmd+R** preview, **Ctrl/Cmd+Shift+R** render, **Alt/Option+R** fast render to MP4.
