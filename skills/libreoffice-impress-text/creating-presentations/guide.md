# Creating presentations (LibreOffice Impress 7.3)

## New Presentation
- Opens with **Select a Template** dialog; click **Cancel** for a blank slide
- To disable template dialog: **Tools > Options > LibreOffice Impress > General** → deselect **Start with Template Selection**

## Inserting a New Slide
New slide is inserted after the selected slide (or as last slide if none selected):
- **Slide > New Slide** on Menu bar
- Right-click in Slides pane → **New Slide**
- Shortcut: `Ctrl+M` (macOS `⌘+M`)
- Right-click in Slide Sorter view → **New Slide**
- Click **New Slide** on Presentation toolbar

## Duplicating a Slide
Duplicate is inserted immediately after the selected slide:
- Right-click in Slides pane → **Duplicate Slide**
- Right-click in Slide Sorter view → **Duplicate Slide**
- **Slide > Duplicate Slide** on Menu bar
- Click **Duplicate Slide** on Presentation toolbar

## Slide Layout
- Default first-slide layout: **Title Slide**; recommended alternatives: **Title Slide** or **Title Only**
- Select layout via **Sidebar > Properties > Layouts panel**, **Slide > Layout**, **Presentation toolbar > Slide Layout**, or right-click slide → **Layouts**
- Impress cannot create custom layouts, but elements can be moved and resized

## Modifying Slide Elements (Normal view only)
- **Move**: click outer frame → cursor changes to clenched hand → drag to new position
- **Resize**: click outer frame → drag selection handle (top/bottom = height; left/right = width; corner = both)
- **Remove**: click element → press `Delete` (macOS `Ctrl+D`) or `Backspace`

## Adding Content
- **Text (contents box)**: click **Click to add Text** → type; outline styles applied automatically
- **Text box**: click **Insert Text Box** on Standard/Drawing toolbar → click slide → type
- **Table**: **Insert > Table** → set columns/rows → **OK**
- **Chart**: **Insert > Chart**
- **Image**: **Insert > Image** → browse and select file
- **Audio/Video**: **Insert > Audio or Video** → browse and select file

## Modifying Slide Appearance
To change background: right-click slide → **Slide Properties** → **Background** → choose type (None, Color, Gradient, Bitmap, Pattern, Hatch) → **OK**

## Reviewing a Presentation
Before finalizing, run the presentation and consider:
- Slide order, spacing/visibility, missing or unnecessary slides
- To hide a slide: right-click in Slides pane → **Hide Slide**
