# Using fields (LibreOffice Impress 7.3)

Fields allow automatic insertion of text into a slide. Commonly used in templates and slide masters.

## Inserting a field

1. Select the slide so it appears in the Workspace.
2. Go to **Insert > Field** and select the field type from the submenu. The field is placed centrally on the slide in a text box.
3. If necessary, position and resize the field text box.
4. If necessary, format the text used for the field information.

## Field types

| Field | Behavior |
|---|---|
| **Date (fixed)** | Inserts current date; not automatically updated |
| **Date (variable)** | Inserts current date; auto-updates each time file is opened |
| **Time (fixed)** | Inserts current time; not automatically updated |
| **Time (variable)** | Inserts current time; auto-updates each time file is opened |

**To change date/time format:** Right-click the field and select the required format from the context menu.

**Note:** Available formats depend on the language setting at **Tools > Options > Language Settings > Language**.
