# Formatting text (LibreOffice Impress 7.3)

## Inserting Formatting Marks
Go to **Insert > Formatting Mark** and select from:
- **Non-breaking space** — `Ctrl+Shift+Space` (macOS `⌘+Shift+Space`)
- **Non-breaking hyphen** — keeps adjacent characters together at line breaks
- **Soft hyphen** — invisible hyphen that appears only when needed at line end
- **Narrow no break space** — ~70% width normal space; `Alt+Shift+Space` (macOS `⌥Shift+Space`)
- **No-width optional break** — `Ctrl+/` (macOS `⌘+/`)
- **Word Joiner** — prevents hyphenation, keeps word together at line end
- **Left-to-right / Right-to-left mark** — requires CTL enabled

## Selecting Text
- **All text in box** — click the text box border
- **Drag** — click and drag over text
- **Word / paragraph** — double-click (word) or triple-click (paragraph)
- **Keyboard** — click, then `Shift+Arrow keys`

## Quick Font Resizing
Select text, then:
- **Increase Font Size** — `Ctrl+]` (macOS `⌘+]`)
- **Decrease Font Size** — `Ctrl+[` (macOS `⌘+[`)

## Styles
- **Presentation Styles** — used in AutoLayout text boxes; cannot be deleted/renamed; change outline level with `Tab` / `Shift+Tab`
- **Drawing Styles** — used in text boxes and graphic objects; Default Drawing Style applied automatically; custom styles available per presentation only

## Paragraph Formatting
1. Select text or paragraph
2. Open **Paragraph** dialog via right-click **> Paragraph** or **Format > Paragraph**
3. Apply options, click **OK**

**Sidebar alternative:** Select text box → **Properties** deck → **Paragraph** panel → format directly; click **More Options** to open full dialog

## Character Formatting
1. Select characters
2. Format via:
   - **Text Formatting toolbar** (immediate)
   - **Format > Character** on Menu bar
   - Right-click **> Character** to open Character dialog → click **OK**
   - **Character** panel in **Properties** deck on Sidebar (immediate)
