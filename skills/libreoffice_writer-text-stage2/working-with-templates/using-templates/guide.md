# Creating Documents from Templates (LibreOffice Writer 7.3.7)

A template is basically a model document — it holds the layout, styles, graphics, and other setup so every new document you create from it starts out looking consistent. Think company letterheads, business reports, or anything you want to reuse without rebuilding from scratch. Templates can also bundle things like custom measurement units, default printer settings, and toolbar configurations.

Every document in LibreOffice is technically based on a template. If you never pick one explicitly, Writer just uses its built-in blank template behind the scenes. You can create, download, or install as many templates as you like, and even set a different default for each document type (text, spreadsheet, drawing, presentation).

To create a new document from a template, you first need to open the Templates dialog. There are several ways to get there — pick whichever feels natural. You can go to **File > New > Templates** or **File > Templates > Manage Templates** from the menu bar. The keyboard shortcut **Ctrl+Shift+N** works too. You can also click the arrow next to the **New** icon on the Standard toolbar and choose **Templates**. If you're on the Start Center with no document open, just select **Templates > Manage Templates** from there.
Once the dialog is open, use the **Filter** tabs at the top to pick the type and category of template you're after. The relevant templates will appear in the main area. You can switch between a thumbnail view and a list view using the icons next to the **Categories** list — thumbnail view is handy for visually browsing, while list view is better when you know what you're looking for.

When you spot the template you want, just double-click it, or select it and click **Open**. Writer creates a brand-new document based on that template, ready for you to edit and save. The template the document is based on gets recorded in **File > Properties > General**, so you can always check later.

One quick tip: from the Start Center, you can also click the **Templates** button or use its drop-down menu to pick a template type directly — this lets you browse and create a new document without even opening the full Templates dialog.

## UI Reference  —  File Menu

_Scope: Templates submenu, New from template_

The File menu manages the full document lifecycle: creating, opening, saving, exporting, printing, and closing documents.

The screenshot shows the File drop-down menu fully expanded in LibreOffice Writer, with the menu bar visible at the top (File, Edit, View, Insert, Format). The menu lists all file-related commands in a single vertical column, organized into logical groups separated by thin dividers: New, Open…, Open Remote…, Recent Documents, and Close at the top; then Wizards and Templates; followed by Reload (greyed out) and Versions…; then the save group (Save, Save As…, Save Remote…, Save a Copy…, Save All with Save All greyed out); the export group (Export…, Export As, Send); Preview in Web Browser and Print Preview (with a checkbox icon); Print… and Printer Settings…; Properties… and Digital Signatures; and finally Exit LibreOffice at the bottom. Some entries such as Save As… and Print Preview show partial keyboard shortcut hints (Shift…) along the right edge of the menu.

## Elements

- **New** (►) — Create a new document of any type (Text, Spreadsheet, Presentation, Drawing, etc.). Ctrl+N for Text Document.
- **Open…** (Ctrl+O) — Open an existing file.
- **Open Remote…** — Open a file from a remote server.
- **Recent Documents** (►) — List of recently opened files.
- **Close** — Close the current document.
- **Wizards** (►) — Step-by-step creation wizards: Letter, Fax, Agenda, Document Converter, Address Data Source.
- **Templates** (►) — Edit Template, Save as Template, Manage Templates (Shift+Ctrl+N).
- **Reload** — Reload document from disk (greyed on unsaved documents).
- **Versions…** — Manage saved versions of the document.
- **Save** (Ctrl+S) — Save in current format.
- **Save As…** (Shift+Ctrl+S) — Save with a new name or format.
- **Save Remote…** — Save to a remote server.
- **Save a Copy…** — Save a copy without changing the working file.
- **Save All** — Save all open documents.
- **Export…** — Export in a non-native format.
- **Export As** (►) — Export as PDF (dialog or direct), Export as EPUB (dialog or direct).
- **Send** (►) — Email document in various formats (ODF, Word, PDF), send via Bluetooth, create Master/HTML document.
- **Preview in Web Browser** — Open document preview in default browser.
- **Print Preview** (Shift+Ctrl+O) — Toggle print-preview layout.
- **Print…** (Ctrl+P) — Open the Print dialog.
- **Printer Settings…** — Configure printer options.
- **Properties…** — Open the 6-tab document properties dialog (General, Description, Custom Properties, Security, Font, Statistics).
- **Digital Signatures** (►) — Digital Signatures…, Sign Existing PDF…
- **Exit LibreOffice** (Ctrl+Q) — Quit the entire suite.
