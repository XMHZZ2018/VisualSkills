# List Types and Basics (LibreOffice Writer 7.3.7)

Writer gives you three flavors of list, and unlike most word processors, it treats list styles as their own separate thing — not just a paragraph style option. That distinction matters because a single list style can pair with multiple paragraph styles, saving you from duplicating design work. You can also swap a paragraph's list style in one click.

**Unordered (bullet) lists** are the ones you'll reach for most often. They're great any time the order of items doesn't matter — think feature lists, notes, or quick summaries. Writer ships with several pre-defined bullet types (round dots, squares, dashes, arrows, and more), and you can pick alternatives from the Gallery or design your own custom bullet graphic.

Writer's pre-defined bullet types are displayed in a two-row, four-column grid of eight thumbnail previews. The top row shows small round dots, large filled circles, filled diamonds, and filled squares; the bottom row shows bold right-pointing arrows, open right-pointing triangles, italic "x" marks, and check marks. Each thumbnail shows three sample list items with placeholder lines next to the bullet character, so you can compare their visual weight before choosing one.

**Ordered (numbered) lists** are what you want when sequence matters — procedures, ranked items, anything where skipping or reordering a point could cause confusion. Each step can contain multiple paragraphs, most of them unordered, describing what happens or listing alternatives. If your procedure grows past a dozen steps, consider breaking it into smaller nested lists. Writer offers pre-defined numbered styles using Arabic numerals, Roman numerals, and letters, and you can define your own.

Writer's pre-defined numbered list styles are displayed in a two-row, four-column grid of eight thumbnail previews. The top row shows four Arabic and Roman numeral formats: "1) 2) 3)", "1. 2. 3.", "(1) (2) (3)", and "I. II. III." The bottom row shows four letter-based formats: "A) B) C)", "a) b) c)", "(a) (b) (c)", and "i. ii. iii." Each thumbnail displays three sample items with placeholder lines, letting you compare the different numbering conventions at a glance.

**Outline lists** summarize the structure of a longer document. In legal and technical writing, they often mirror heading levels to make hierarchy obvious. Writer provides several outline options that use a single paragraph style — you control the level by pressing **Tab** to go deeper and **Shift+Tab** to move back up. This single-style approach is by far the quickest to learn and apply. You can also strip the numbering later by changing the associated style, leaving a clean document for publication.

One handy tip: the default toolbar icons for lists apply a fixed style you can't change from the icon itself. If you want more control, create paragraph styles for your lists and, optionally, add custom toolbar buttons for them (see **Tools > Customize** for toolbar editing).

## UI Reference  —  Formatting Toolbar

_Scope: Toggle Unordered/Ordered List buttons, Select Outline Format_

The second toolbar row provides all character and paragraph formatting controls with split-button dropdowns.

The Formatting toolbar is a single horizontal row of controls on a dark gray background. From left to right it contains: a "Paragraph Style" dropdown (currently showing "Default Paragraph Style"), two small style-update icons, a "Font Name" dropdown (showing "Liberation Serif"), and a "Font Size" dropdown (showing "12 pt"). After a separator, the toolbar continues with Bold (B), Italic (I), and Underline (U) buttons — the Underline button has a small dropdown arrow — followed by Strikethrough (S), Superscript (x²), and Subscript (x₂) buttons. Next come a Font Color button (showing an "A" with a colored underline and a dropdown arrow) and a Character Highlighting Color button (also with a dropdown arrow). The visible portion of the toolbar ends with alignment buttons for left, center, right, and justified alignment; the list-toggle and indent buttons extend further to the right beyond the visible area.

## Elements

Row (left → right):

- **Paragraph Style** dropdown — Shows current style (e.g. "Default Paragraph Style"). Dropdown lists: Clear formatting, Default Paragraph Style, Body Text, Title, Subtitle, Heading 1–4, Block Quotation, Preformatted Text, More Styles…
- **Update Selected Style** (Shift+Ctrl+F11) — Update current style to match cursor formatting.
- **New Style from Selection** (Shift+F11) — Create a new style from current formatting.
- **Font Name** dropdown — Shows/changes the font (e.g. "Liberation Serif"). Lists all installed fonts rendered in their own typeface.
- **Font Size** dropdown — Shows/changes size in pt (6–48, plus custom values).
