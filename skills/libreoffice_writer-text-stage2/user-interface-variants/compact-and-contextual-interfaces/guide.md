# Compact and Contextual Interfaces (LibreOffice Writer 7.3.7)

LibreOffice Writer offers several space-saving interface layouts that put tools closer to your fingertips without eating up your screen. You can switch between them anytime via **View > User Interface** from the standard menu bar, or — if you're already using the Groupedbar layout — through the **Menu** drop-down on the bar itself.

## Tabbed Compact

The Tabbed Compact interface keeps the same familiar tabs as the full Tabbed layout (Home, Insert, Layout, References, and so on), but collapses everything into a single row of icons per tab. Most of those icons have small drop-down arrows that expand into menus of related commands. It's a great middle ground: you still get the tabbed organization, but with a much thinner toolbar area.

## Groupedbar Compact

The Groupedbar Compact interface takes a different approach. Instead of tabs, it divides commands into labeled groups — things like **Text Body Indent**, **Liberation Sans**, font size, bold/italic/underline — all visible at once. Below those groups sit a row of tab-like buttons (**Insert**, **References**, **Review**, **View**, **Form**, **Extension**) that swap the visible command set.

The Groupedbar also has its own **Menu** button on the far right. Clicking it gives you quick access to **Menubar**, **Sidebar** (**Ctrl+F5**), **User Interface**, **Toolbars**, **Customize**, and **Options** (**Alt+F12**) — essentially a shortcut to all the configuration you'd normally reach through the menu bar.

Above the menus, the Groupedbar's bottom row of tab-like buttons (View, Form, Extension, Menu) is visible along with their associated toolbar icons.

## Contextual Single

The Contextual Single interface is the most minimal option. It shows a standard menu bar plus just one toolbar whose contents change depending on what you're working with — select a table and you'll see table tools, click into normal text and you'll get formatting options. If you want maximum document space with minimal chrome, this is the layout to pick.

To try any of these, just head to **View > User Interface**, pick the variant you want, and Writer switches instantly. No restart needed — experiment freely until you find what fits your workflow.
