# Tabbed Interface (LibreOffice Writer 7.3.7)

The Tabbed interface is Writer's answer to the ribbon-style layout found in Microsoft Office. It swaps out the traditional toolbar stack for a row of grouped tabs, each packed with contextually relevant icons. To switch to it, head to **View > User Interface** and select **Tabbed**.

The interface consists of a few key pieces: an **Icon bar** at the top left (with shortcuts for open, save, undo, redo, and print), a **Tab bar** across the top, icons on the **active tab** itself, and a **Quick menu** at the far right. On Windows and Linux, you can toggle the Menu bar on or off by clicking the Menu Bar icon in the Icon bar.

The Tabbed interface is shown with the Menu bar hidden and the File tab active. The Icon bar on the far left contains new, open, save, undo, redo, and print icons. Across the top, the tab row displays File, Home, Insert, Layout, References, Review, View, Extension, and Tools tabs. The File tab's content area shows icons for Templates, New, Save as Template, Recent Documents, Open, Open Remote, Save As, Save, Save Remote, Export to PDF, Export to EPUB, Print Directly, Print, Print Preview, Digital Signatures, Sign Existing PDF, and Properties. At the far right, a Tab menu drop-down arrow and the Quick menu (hamburger icon) are visible, along with File and Help drop-down menus.

If the icons on a tab don't all fit in the window width, a **>>** chevron appears at the end of the row — just click it to reveal the rest.

## Fixed tabs

Writer gives you nine fixed tabs that are always available: **File**, **Home**, **Insert**, **Layout**, **References**, **Review**, **View**, **Extension**, and **Tools**. The **File** tab is your hub for creating, opening, saving, exporting to PDF/EPUB, managing templates, and accessing document properties. It also has its own **File** and **Help** drop-down menus right on the tab.

The **Home** tab is where you'll spend most of your time — it covers cut/copy/paste, font and paragraph formatting, style selection, and inserting common items like images, tables, and special characters. The **Insert** tab goes deeper with page breaks, sections, charts, OLE objects, media, bookmarks, hyperlinks, cross-references, and fields.

**Layout** handles page-level formatting: margins, orientation, page size, columns, and page breaks. You'll also find watermark, hyphenation, and text wrap controls here. The **References** tab is purpose-built for academic and long-document work — table of contents, footnotes, endnotes, index entries, bibliographies, and cross-references all live here.

The Home tab is shown in two rows. The upper row contains Paste, Cut, Copy, Clone, and Clear formatting buttons on the left, followed by a font name drop-down (set to "Liberation Sans"), a font size field (set to "11 pt"), font grow/shrink buttons, spacing controls, and paragraph alignment and indent icons, ending with a paragraph mark toggle. The lower row displays style previews including Text Body, Heading 1, Heading 2, Heading 3, Heading 4, and Title, along with navigation arrows, a Table drop-down, image/chart/comment insert icons, a Zoom control, and a Find button. A "Home" drop-down menu is visible at the far right of the lower row.

**Review** gives you quick access to spelling, thesaurus, word count, and tracked changes (record, show, accept, reject, compare, and merge). **View** lets you switch between Normal, Web, and Print Preview, toggle formatting marks, rulers, the sidebar, and the navigator. **Extension** is minimal — it just provides access to the **Extension Manager** (Ctrl+Alt+E). **Tools** rounds things out with macros, mail merge, envelope creation, form controls, and design mode toggling.

## Additional (contextual) tabs

Beyond the nine fixed tabs, Writer displays additional tabs when you select certain objects. Select a drawing object and a **Draw** tab appears with captioning, line, color, and anchor controls. Click on an image and the **Image** tab shows up with cropping, borders, area styles, filtering, and wrap options. Likewise, **Media** appears for audio/video files, **Object** for frames and embedded objects, and **Table** when your cursor is inside a table.

Each tab also has a small drop-down menu on its label (e.g., clicking the **Home** drop-down) that provides additional commands not shown as icons on the tab itself — handy for less common operations.

The contextual Table tab is shown, which appears when the cursor is inside a table. The upper row includes Caption, border style icons, insert/delete row and column buttons, Merge Cells, Split Cells, Split Table, Select Cell, cell selection grid icons, Table Styles, text wrap controls, and table border formatting options. The lower row contains Optimize Size, vertical alignment buttons (top, center, bottom, justify), Sort, Sum, Formula, Number Format, percentage and decimal formatting icons, and at the far right a "Table" drop-down menu and a Table Properties button. The tab row shows that Table appears as an additional tab alongside the standard fixed tabs.

You can customize which icons appear on each tab through **View > User Interface**, then using the **Notebookbar** tab of the Customize dialog. This lets you show, hide, or rearrange individual icons to match your workflow.
