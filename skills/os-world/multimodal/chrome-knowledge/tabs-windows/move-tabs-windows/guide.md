# Move Tabs Between Windows

1. Open multiple Chrome windows by pressing Cmd+N (Mac) or Ctrl+N (Windows) to create a new window alongside your existing one.
2. Click and hold on the tab you want to move. The tab will lift and become draggable.

   ![Step 2](step02.png)

3. Drag the tab out of the current window's tab strip and into the tab strip of the other Chrome window to move it there.

   ![Step 3](step03.png)

4. To move a tab to a brand new window, drag it completely away from all existing windows and release it — Chrome will open it as a standalone window.

   ![Step 4](step04.png)

5. Alternatively, right-click the tab and select 'Move tab to new window' to instantly detach it into its own window without dragging.

   ![Step 5](step05.png)

6. To move multiple tabs at once, hold Ctrl (Windows) or Cmd (Mac) and click each tab to select them, then drag the group to another window.
