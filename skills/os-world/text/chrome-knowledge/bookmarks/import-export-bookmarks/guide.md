# Import and Export Bookmarks

## Export Bookmarks

1. Click the three-dot menu (⋮) in the top-right corner of Chrome.
2. Navigate to **Bookmarks > Bookmark Manager** (or go to `chrome://bookmarks`).
3. Click the three-dot menu inside Bookmark Manager.
4. Select **Export bookmarks**.
5. Choose a destination (e.g., Desktop) and click **Save**. A `.html` file will be saved.

## Import Bookmarks

1. Click the three-dot menu (⋮) in the top-right corner of Chrome.
2. Navigate to **Bookmarks > Import bookmarks and settings**.
3. In the dropdown, select **Bookmarks HTML file**.
4. Click **Choose File**, locate your saved `.html` bookmarks file, and click **Open**.
5. Your bookmarks are now imported and visible in Chrome.
