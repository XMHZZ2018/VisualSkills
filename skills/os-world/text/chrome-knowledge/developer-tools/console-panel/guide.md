# Use the Console Panel

1. Open Chrome DevTools by pressing `Cmd+Option+J` (Mac) or `Ctrl+Shift+J` (Windows/Linux) — this opens directly to the Console panel.
2. Alternatively, press `F12` or right-click any page element and select **Inspect**, then click the **Console** tab.
3. View logged messages, warnings, and errors that appear automatically as the page loads and runs.
4. Click on any error or warning message to expand its stack trace and see where it originated.
5. Use the filter buttons (All, Errors, Warnings, Info, Verbose) at the top to narrow down log output.
6. Type JavaScript directly into the input prompt at the bottom (marked with `>`) and press `Enter` to execute it.
7. Run multi-line code by pressing `Shift+Enter` to add new lines without executing.
8. Access previously run commands by pressing the `Up Arrow` key to cycle through history.
9. Use `console.log()`, `console.warn()`, `console.error()`, and `console.table()` in your code to output values.
10. Click the **Clear console** button (🚫 icon) or type `clear()` and press `Enter` to clear all output.
11. Right-click any logged object to **Store as global variable** for further inspection in the prompt.
12. Use the **Live Expressions** button (eye icon) to pin expressions that update in real time as the page runs.
