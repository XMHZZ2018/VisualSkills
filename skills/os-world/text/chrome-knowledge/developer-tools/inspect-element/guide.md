# Inspect and Edit Page Elements

1. Open Chrome and navigate to any webpage you want to inspect.
2. Right-click anywhere on the page and select **Inspect** (or press `F12` / `Cmd+Option+I` on Mac).
3. Click the **Elements** panel tab at the top of DevTools.
4. Click the element picker icon (top-left of DevTools, cursor with box) or press `Ctrl+Shift+C` / `Cmd+Shift+C` to activate inspect mode.
5. Hover over the page and click any element to select it in the Elements panel.
6. Double-click the HTML tag or attribute in the Elements panel to edit it directly.
7. Press `Enter` to confirm the change — it applies live to the page.
8. To edit CSS, find the element's styles in the **Styles** pane on the right side of the Elements panel.
9. Click any CSS value to edit it, or click the empty space in a rule to add a new property.
10. To add a new CSS rule, click the `+` icon in the Styles pane.
11. To delete an HTML element, select it in the Elements panel and press `Delete`.
12. To hide an element without deleting it, select it and press `H`.
13. Right-click an element in the Elements panel and choose **Edit as HTML** to edit the full HTML block.
14. Close DevTools — changes are temporary and reset on page refresh.
