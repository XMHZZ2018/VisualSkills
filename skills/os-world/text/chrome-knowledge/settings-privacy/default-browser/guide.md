# Set Chrome as Default Browser

## Windows 10/11

1. Open the **Start menu** and click the gear icon to open **Settings**
2. Click **System**
3. Click **Default apps** in the left sidebar
4. Scroll down to **Web browser** and click the currently set browser
5. Select **Google Chrome** from the list
6. If prompted, click **Switch anyway** to confirm

## macOS

1. Open **System Settings** (Apple menu > System Settings)
2. Click **Desktop & Dock** or search for **Default web browser**
3. Under **Default web browser**, click the dropdown and select **Google Chrome**

## From Within Chrome

1. Navigate to `chrome://settings/`
2. Click **Make default** at the top of the Settings page (if Chrome is not already the default)
3. Follow the OS prompt to confirm
