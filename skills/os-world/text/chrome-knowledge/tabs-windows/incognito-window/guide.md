# Open Incognito Window

1. **Keyboard shortcut (fastest):** Press `Ctrl+Shift+N` (Windows/Linux) or `Cmd+Shift+N` (Mac).

2. **From the Chrome menu:** Click the three-dot menu (⋮) in the top-right corner, then select **New Incognito Window**.

3. **From the taskbar (Windows):** Right-click the Chrome icon in the taskbar and select **New Incognito Window**.

**What to expect:**
- The incognito window has a dark/black theme to distinguish it from regular windows.
- Chrome will not save your browsing history, cookies, or form data from this session.
- Downloads and bookmarks made during the session are still saved.
- Closing all incognito windows ends the private session and clears session data.
