# Create a Chrome Profile

1. Open Chrome and click the profile icon (circle with your initial/photo) in the top-right corner of the browser window.
2. In the dropdown, scroll down and click **Add** under "Other profiles".
3. Enter a name for the new profile (e.g., "Home", "Work").
4. Optionally, choose a profile photo and theme color.
5. Optionally, check **Create a desktop shortcut** to launch this profile directly from your desktop.
6. Click **Done** — Chrome will open a new window for the new profile.
7. To switch profiles, click the profile icon in the top-right and select another profile from the dropdown.
8. To manage or delete profiles, navigate to `chrome://settings` and click **You and Google** > **Manage other people**.
