# Manage Extension Permissions

1. Navigate to `chrome://extensions` in the address bar.
2. Find the extension you want to manage and click **Details**.
3. Scroll to the **Site access** section to view current permissions.
4. Change site access using the dropdown: select **On click**, **On specific sites**, or **On all sites**.
5. To restrict to specific sites, select **On specific sites**, then click **Add a new site** and enter the URL.
6. Scroll down to review other permissions listed under **Permissions** (read-only list of what the extension can access).
7. To remove the extension entirely, click **Remove** on the extensions page.
8. Alternatively, right-click the extension icon in the toolbar and select **Manage extension** to jump directly to its details page.
