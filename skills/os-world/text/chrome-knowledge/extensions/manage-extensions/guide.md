# Enable, Disable, and Remove Extensions

## Via the Toolbar (Quick Remove)
1. Right-click the extension icon in the top-right toolbar.
2. Select **Remove from Chrome**.

## Via Extensions Manager
1. Click the three-dot menu (⋮) in the top-right corner of Chrome.
2. Go to **More tools** > **Extensions**, or navigate to `chrome://extensions`.
3. Find the extension you want to manage.
4. **To disable:** Toggle the blue switch to off.
5. **To enable:** Toggle the switch back on.
6. **To remove:** Click **Remove**, then confirm by clicking **Remove** in the dialog.
7. If prompted with a feedback website, close it — the extension is already uninstalled.
