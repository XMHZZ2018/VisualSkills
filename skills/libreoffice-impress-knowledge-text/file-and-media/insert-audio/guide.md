# Insert Audio (LibreOffice Impress 7.3.7)

## Method 1: File Browser

1. Go to **Insert > Audio or Video** on the Menu bar.
2. Navigate to the folder containing your audio file. Only compatible files will be available.
3. Select the file and click **Open** — it embeds in the center of the slide. The **Media Playback** toolbar opens automatically.
4. Reposition and/or resize as needed.

## Method 2: Gallery

1. Open the Gallery deck using one of:
   - **View > Sidebar**, then select **Gallery**
   - Keyboard shortcut **Ctrl+F5**, then select **Gallery**
   - **View > Gallery**
2. Browse to a theme containing media files (e.g., **Sounds**).
3. Click the audio option and drag it onto the slide. The **Media Playback** toolbar opens automatically.
4. Reposition and/or resize as needed.

## Media Playback Toolbar

Opens automatically when an audio file is selected. Controls (left to right):

| Tool | Function |
|------|----------|
| **Insert Audio or Video** | Opens the file browser to insert media |
| **Play** | Plays the media |
| **Pause** | Pauses playback |
| **Stop** | Stops playback |
| **Repeat** | Loops playback until deselected |
| **Position** | Sets the start position within the file |
| **Mute** | Suppresses volume |
| **Volume** | Adjusts volume |
| **Media Path** | Shows where the file is stored on disk |
