# Export Slides as Images (LibreOffice Impress 7.3.7)

Export slides as image files using the **Export in Other Formats** method:

1. Open your presentation in Impress.
2. Go to **File > Export** on the menu bar to open a file browser window.
3. Navigate to the folder where the image files will be saved.
4. Enter a name for the exported file in the **Name** text box.
5. At the bottom of the file browser window, select the required image format (e.g., PNG) from the drop-down list.
6. Click **Save**. An options dialog may open for further format-specific settings.
7. Click **OK** to save the exported file.

> **Note:** For more information on exporting files, see the Getting Started Guide.
