# Save / Save As (LibreOffice Impress 7.3.7)

## Save a New Presentation

1. Create your slide show.
2. Open the save dialog using one of:
   - Click **Save** on the Standard toolbar
   - Go to **File > Save**
   - Press **Ctrl+S** (macOS **⌘+S**)
3. Navigate to the destination folder.
4. Enter a filename in the **File name** / **Name** / **Save as** text box.
5. Select a file format from the drop-down list (default: **ODF Presentation (.odp)**).
6. Click **Save**.

> If saving in a format other than ODP, a **Confirm File Format** dialog will appear asking you to confirm the selected format.

---

## Save to ODP Format (Existing File)

Use any of the following:
- Click **Save** on the Standard toolbar
- Go to **File > Save**
- Press **Ctrl+S** (macOS **⌘+S**)

---

## Save As Another Format (e.g., .pptx)

1. Open the Save As dialog using one of:
   - Click the triangle **▼** to the right of **Save** on the Standard toolbar and select **Save As**
   - Go to **File > Save As**
   - Press **Ctrl+S** (macOS **⌘+Shift+S**)
2. Navigate to the destination folder.
3. Enter a filename in the **Name** text box.
4. Select the desired format from the drop-down list (default: **ODF Presentation (.odp)**).
5. Click **Save**.
