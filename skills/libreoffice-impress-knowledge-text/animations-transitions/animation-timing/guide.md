# Set Animation Timing and Order

1. Select all slides to include: click the first slide, hold **Shift**, then click the last slide.
2. Click **Slide Show > Animation** (or **Transitions** tab) in the menu bar.
3. Choose the desired transition effect from the available options.
4. In the **Advance Slide** section (far right), uncheck **On Mouse Click**.
5. Check **Automatically After** and enter the desired display duration (e.g., `00:01` for 1 second).
6. To set animation order and timing for objects within a slide, open **View > Animation** panel.
7. Select an animated object and click **Add Effect** to assign an animation.
8. In the Animation panel, click an effect to select it, then use **Effect Options** to set the delay and duration.
9. Drag effects up or down in the Animation panel list to reorder the sequence.
10. Set **Start** to **On Click**, **After Previous**, or **With Previous** to control trigger behavior.
11. Run the slideshow via **Slide Show > Start from First Slide** to verify timing and order.
