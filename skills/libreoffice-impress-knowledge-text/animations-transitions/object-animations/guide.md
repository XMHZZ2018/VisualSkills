# Add Object Animations

1. Select the object (text box, image, or shape) you want to animate.
2. Navigate to **Slideshow > Animation** to open the Animation panel.
3. Click **Add** in the Animation panel.
4. In the dialog, choose a category from the **Category** dropdown:
   - **Entrance** — object appears on screen
   - **Exit** — object leaves the screen
   - **Emphasis** — object draws attention while staying on screen
5. Browse the effect list and click an effect to preview it.
6. Click **OK** to apply the effect.
7. Set the **Start** trigger at the bottom of the panel:
   - **On Click** — plays when you click during the slideshow
   - **With Previous** — plays simultaneously with the prior animation
   - **After Previous** — plays automatically after the prior animation ends
8. Repeat steps 1–7 for each object you want to animate.
9. Use the animation list in the panel to reorder effects by dragging them up or down.
10. Click **Play** in the Animation panel to preview all animations on the current slide.
