# Insert and Edit Tables

1. Go to **Insert > Table** (or press **Ctrl+F12**) to open the Insert Table dialog.
2. Set the number of **Columns** (left-to-right) and **Rows** (top-to-bottom), then click **OK**.
3. Click any cell and start typing to enter content.
4. To add a row: click the far-left edge to select a row, right-click, then choose **Insert > Rows Above** or **Rows Below**.
5. To add a column: click the top edge to select a column, right-click, then choose **Insert > Columns Left** or **Columns Right**.
6. To format a header row: click the left edge to select the entire first row.
7. With the row selected, go to **Table > Table Properties** to apply background color, borders, or other formatting.
8. To change cell background: select the cell(s), right-click, choose **Table Properties**, and set the background color under the **Background** tab.
9. To merge cells: select multiple adjacent cells, right-click, and choose **Merge Cells**.
10. To adjust column width or row height: click and drag the cell borders, or use **Table > Table Properties > Columns/Rows** for precise values.
