# Run a Slideshow

1. Open your presentation in LibreOffice Impress.
2. Press **F5** to start the slideshow from the first slide, or press **Shift+F5** to start from the current slide.
3. Click the mouse or press **Space** / **→** (right arrow) to advance to the next slide.
4. Press **←** (left arrow) or **Backspace** to go back to the previous slide.
5. Press a slide number followed by **Enter** to jump directly to that slide.
6. Press **Esc** to end the slideshow and return to the editing view.

**Alternative start methods:**
- Go to **Slide Show > Start from First Slide**
- Go to **Slide Show > Start from Current Slide**
