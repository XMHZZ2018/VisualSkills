# Edit Table Content (LibreOffice Impress 7.3.7)

## Entering and Editing Cell Text

Click inside a cell to position the cursor, then type to enter or edit text.

## Navigating Cells

Use Tab to move between cells while editing.

## Text Alignment in Cells

**Vertical alignment** — select cells, then use one of:
- **Align Top**, **Center Vertically**, or **Align Bottom** on the Table toolbar
- Right-click > **Align**
- **Format > Align Text** on the Menu bar

## Merging and Splitting Cells

**Merge:** Select cells > **Merge Cells** on Table toolbar, or right-click > **Merge Cells**, or **Format > Table > Merge Cells**

**Split:** Click in cell > **Split Cells** on Table toolbar to open the Split Cells dialog. Choose number of cells and direction (**Horizontally** or **Vertically**). When splitting horizontally, use **Into equal proportions** for equal-sized cells. Contents remain in the original (left or top) cell.

## Font and Text Formatting

Open **Table Properties** dialog via:
- Right-click in table > **Table Properties**
- **Format > Table > Properties**
- **Table Properties** button on Table toolbar

| Tab | Options |
|-----|---------|
| **Font** | Family, Style, Size, Language |
| **Font Effects** | Font Color, Text Decoration, Effects |

## Optimizing Cell Size

Click **Optimize** on the Table toolbar to access:
- **Minimal/Optimal Column Width** — based on shortest/longest column entry
- **Minimal/Optimal Row Height** — based on font size
- **Distribute Columns Evenly** / **Distribute Rows Equally**

Also available by right-clicking selected cells > appropriate **Optimize** option.
