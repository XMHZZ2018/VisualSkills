# Insert a Table (LibreOffice Impress 7.3.7)

## Method 1: Insert Table Dialog

1. Select the slide and modify the layout if needed to create space.
2. Go to **Insert > Table** to open the Insert Table dialog.
3. Specify the **Number of columns** and **Number of rows**.
4. Click **OK** to insert the table.

## Method 2: Table Graphic

1. Select the slide and modify the layout if needed.
2. Click **Table** on the Standard or Table toolbar to open the Table graphic.
3. Click, hold, and drag across and down to select the required columns and rows.
4. Release to insert the table. Alternatively, click **More Options** in the Table graphic to open the Insert Table dialog.

> **Note:** The table is created inside a text box at the center of the slide with default style and settings applied. The text box is the same size as the rows and columns of the table.

## After Inserting: Basic Formatting

**Table Design (Sidebar)**
- With a table selected, click **Table Design** in the Properties deck on the Sidebar.
- Choose a design and configure row/column options:
  - **Header row** – first row displayed with different background (selected by default)
  - **Total row** – highlights last row
  - **Banded rows** – alternating row backgrounds (selected by default)
  - **First column / Last column** – highlights with darker background
  - **Banded columns** – alternating column highlights

**Table Toolbar** (opens automatically when a table is selected; if not visible, go to **View > Toolbars > Table**)
- **Merge Cells** – merges selected cells; or right-click > **Merge Cells**, or **Format > Table > Merge Cells**
- **Split Cells** – splits a cell horizontally or vertically; choose number of cells and direction in the Split Cells dialog
- **Insert Row Above / Below, Insert Column Before / After** – select a row/column then click the tool; or **Format > Table > Insert Rows / Insert Columns**
- **Delete Row / Delete Column / Delete Table** – select a cell and click the tool; or **Format > Table > Delete Row / Delete Column / Delete Table**
- **Table Properties** – opens dialog with tabs for **Font**, **Font Effects**, **Borders**, **Background**, and **Shadow**; also accessible via right-click > **Table Properties** or **Format > Table > Properties**
