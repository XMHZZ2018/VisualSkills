# Move / Reposition a Table (LibreOffice Impress 7.3.7)

Tables in Impress are placed inside text boxes and treated like any other graphic object.

## Drag to Reposition

Click on the border of the table and drag to move the table text box to a new position.

## Position and Size Dialog (Precise Placement)

Open the **Position and Size** dialog using one of these methods after selecting the table:

- Right-click the table and select **Position and Size** from the context menu
- Go to **Format > Object and Shape > Position and Size** on the Menu bar
- Press **F4**

> **Note:** Only the Position and Size dialog can be used for precise positioning of tables (not other positioning tools available for regular objects).

## Resizing (affects position)

Click and drag the selection handles on the table text box to resize it. When the text box size changes, the table also increases or decreases in size to keep the text box filled.
