# Add Speaker Notes (LibreOffice Impress 7.3.7)

Speaker notes are not displayed during a presentation, but can be shown on a second display using the Presenter Console.

## Adding Notes

1. Select a slide to add notes to, then open Notes view using one of:
   - Click **Notes** at the top of the Workspace
   - Go to **View > Notes** on the Menu bar
   - Click **Display Views** and select **Notes** in Edit Modes on the Standard toolbar
2. Click **Click to add notes** below the slide, then type or paste text as required.
3. Repeat for additional slides.
4. When done, return to Normal view.

## Formatting Notes (via Master Notes)

Use Master Notes to format all notes consistently rather than slide by slide.

1. Select a slide that has notes, then open Master Notes view using one of:
   - Go to **View > Master Notes** on the Menu bar
   - Click **Master** on the Standard toolbar, then click **Notes** at the top of the Workspace
   - Click **Display Views** and select **Master Notes** in Master Modes on the Standard toolbar
2. Open the Slide Properties dialog via **Slide > Slide Properties** or right-click an empty area and select **Slide Properties**.
3. Configure options and click **OK**.

## Printing Notes

1. Open Notes view for a slide that has notes.
2. Open the Print dialog: **File > Print** or `Ctrl+P` (macOS: `⌘P`).
3. **Linux/Windows:** Select the **LibreOffice Impress** tab → set **Type** to **Notes**.
4. **macOS:** Select **LibreOffice** from the drop-down → set **Document** to **Notes**.
5. Select any additional options and click **OK**.

## Exporting Notes as PDF

The exported PDF contains slide pages followed by notes pages (e.g., 10 slides → 20 PDF pages).

1. Open Notes view for a slide that has notes.
2. Go to **File > Export As > Export as PDF**.
3. On the **General** page, select **Export notes pages**.
4. Optionally select **Export only notes pages** for a notes-only PDF.
5. Click **Export**, choose a save location, enter a filename, and click **Save**.
