# Slide Number Formatting (LibreOffice Impress 7.3.7)

## Change the Number Format (1, 2, 3 / a, b, c / i, ii, iii)

Go to **Slide > Slide Properties** on the Menu bar to open the Slide Properties dialog. In **Layout Settings**, select the type of number format from the **Slide numbers** drop-down list.

## Change the Appearance (Color, Font, etc.) of a Slide Number

Slide numbers appear as fields in the footer's right section or as manually inserted fields. To format them:

1. Go to **View > Master Slide** on the Menu bar to open master slide view.
2. Select the master slide so it appears in the Workspace.
3. Click on the slide number field to select it.
4. Format the text using one of the following methods:
   - Use the formatting tools on the **Text Formatting toolbar**.
   - Go to **Format** on the Menu bar and select a formatting option.
   - Use the options in the **Character** dialog.
   - Use the **Character** panel in the Properties deck on the Sidebar.
5. Click **Close Master View** on the Master View toolbar, or go to **View > Normal**.

## Insert a Slide Number Field (Manual)

1. Go to **View > Master Slide** on the Menu bar.
2. Select the master slide.
3. Click anywhere on the master slide without selecting an object.
4. Go to **Insert > Field** on the Menu bar and select **Slide number** from the context menu.
5. Reposition the field text box as needed.
6. Format the text as described above.
7. Click **Close Master View** or go to **View > Normal**.

## Replace the Default Slide Number Footer Field

1. Go to **View > Master Slide** on the Menu bar.
2. Select the master slide.
3. Highlight all characters of the default `<number>` field in the footer's right section.
4. Go to **Insert > Field** and select **Slide number**.
5. Format the field text as needed.
6. Click **Close Master View** or go to **View > Normal**.
