# Change Slide Orientation (LibreOffice Impress 7.3.7)

## Via Sidebar (changes apply immediately)

1. Make sure **Normal** view is selected in the Workspace.
2. Click **Properties** in the Sidebar to open the Properties deck.
3. Click **Slide** to open the Slide panel in the Properties deck.
4. Select the slide orientation from the **Orientation** drop-down list.

## Via Slide Properties Dialog

1. Make sure **Normal** view is selected in the Workspace.
2. Open the Slide Properties dialog using one of the following methods:
   - Go to **Slide > Slide Properties** on the Menu bar.
   - Right-click in an empty area on the slide and select **Slide Properties** from the context menu.
3. Use the options in the **Slide** page to set the orientation.
4. Click **OK** to save the changes and close the dialog.
