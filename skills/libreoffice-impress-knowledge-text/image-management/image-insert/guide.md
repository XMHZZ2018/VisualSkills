# Insert an Image (LibreOffice Impress 7.3.7)

## From a File

1. Select or insert a slide.
2. Go to **Insert > Image** — the Insert Image file browser opens.
3. Navigate to and select the image file.
4. Optionally enable **Preview** to see a thumbnail.
5. Click **Open** — the image is placed in the center of the slide with selection handles.

**To link instead of embed:** Select **Insert as Link** in the file browser. Use linking when the file is large, shared across presentations, or only available at runtime.

---

## From the Gallery (Sidebar)

1. Open the Gallery deck via **Insert > Media > Gallery** or click **Gallery** in the Sidebar.
2. Select a theme — its images are displayed.
3. Scroll to find the required image.
4. Click, hold, and drag the image onto the slide.

---

## From a Scanner

1. Prepare and power on the scanner.
2. First use: go to **Insert > Media > Scan > Select Source** to open the scanner dialog.
3. Subsequent use: go to **Insert > Media > Scan > Request**.
4. If multiple scanners are connected, select one from the **Select Source** or **Device Used** drop-down.
5. Click **Create Preview** or **Preview** if available, then crop and adjust as needed.
6. Click **Scan** — the image is placed in the center of the slide.

---

## Managing Gallery Themes

**Create a theme:** Open Gallery → click **New** → enter a name on the **General** page → click **OK**.

**Add images to a custom theme:** Right-click the theme → **Properties** → **Files** → **Find Files** → select a folder → select files → click **Add** (or **Add All**).

**Delete an image from a custom theme:** Right-click the image → **Delete** → confirm. *(Original files are not deleted.)*

**Update a custom theme:** Right-click the theme name → **Update** to refresh all file links.

> Default themes are read-only; only custom themes support adding, deleting, or updating images.
