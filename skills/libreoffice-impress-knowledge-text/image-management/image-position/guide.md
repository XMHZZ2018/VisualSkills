# Move / Reposition an Image (LibreOffice Impress 7.3.7)

## Quick Move (Cursor)

1. Click the image to display selection handles.
2. Move the cursor over the image until the cursor changes shape.
3. Click and drag to the desired position. A ghost image appears to help with placement.
4. Release when in position.

> **Tip:** Arrow keys can also nudge a selected object to a new position.

---

## Precise Placement (Position and Size Dialog)

1. Select the image, then open the Position and Size dialog via one of:
   - Keyboard shortcut **F4**
   - **Format > Object and Shape > Position and Size**
   - Right-click the object > **Position and Size**
   - **More Options** in the Position and Size panel on the Sidebar
2. Click the **Position and Size** tab.
3. In the **Position** section, enter **Position X** (horizontal) and **Position Y** (vertical). Values are relative to the selected **Base point** (default: top-left corner of the slide).
4. Click **OK**.

> **Note:** Units of measurement are set in **Tools > Options > LibreOffice Impress > General** (macOS: **LibreOffice > Preferences > LibreOffice Impress > General**).

---

## Precise Placement (Sidebar)

1. Select the image.
2. Click **Properties** in the Sidebar to open the Properties deck.
3. Click **Position and Size** to open the panel.
4. Enter **Position X** and **Position Y** values (relative to the Base point set in the Position and Size dialog).
5. Press **Enter** to apply.

---

## Lock Position (Prevent Accidental Moving)

In the Position and Size dialog (**F4**), under **Protect**, select the **Position** option to prevent repositioning.
