# Resize an Image (LibreOffice Impress 7.3.7)

## Method 1: Drag to Resize

1. Click the image to select it and display the selection handles.
2. Move the cursor over a selection handle — it changes shape to indicate resize direction.
3. Click and drag the handle to resize. A ghost image appears to guide resizing.
4. Release when the image is at the desired size.

**Handle behavior:**
- Corner handles — resize width and height simultaneously
- Top/bottom handles — resize height only
- Left/right handles — resize width only

**To maintain proportions:** Hold **Shift** before clicking and dragging a handle. Release the mouse button before releasing **Shift**.

> **Tip:** Resizing raster images causes blurring. Scale images in a dedicated graphics program (e.g., GIMP) before inserting them into a presentation.

---

## Method 2: Position and Size Dialog (Precise)

1. Select the image, then open the **Position and Size** dialog using one of:
   - Keyboard shortcut **F4**
   - **Format > Object and Shape > Position and Size**
   - Right-click the image and select **Position and Size**
   - Click **More Options** in the Position and Size panel on the Sidebar
2. Click the **Position and Size** tab.
3. In the **Size** section, select a **Base point** to anchor during resizing (default: top left corner).
4. To maintain proportions, enable **Keep ratio** before changing **Width** or **Height** — adjusting one dimension automatically updates the other.
5. Enter the desired **Width** and **Height**.
6. Optionally, in the **Protect** section, enable **Size** to prevent future resizing.
7. Click **OK**.

---

## Method 3: Sidebar Panel

1. Select the image to display the selection handles.
2. Click **Properties** in the Sidebar to open the Properties deck.
3. Click **Position and Size** to open the panel.
4. To maintain proportions, enable **Keep ratio** before changing **Width** or **Height**.
5. Enter the desired dimensions and press **Enter**.
