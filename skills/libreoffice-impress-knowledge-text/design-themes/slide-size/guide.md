# Change Slide Size and Orientation

1. Go to **Format > Page** (or **Slide > Slide Properties** in newer versions).
2. In the **Page Setup** dialog, locate the **Width** and **Height** fields.
3. To set 16:9 widescreen, enter **Width: 25.4 cm (10 in)** and **Height: 14.29 cm (5.63 in)**, or use the preset dropdown if available.
4. To set 4:3 standard, enter **Width: 25.4 cm (10 in)** and **Height: 19.05 cm (7.5 in)**.
5. To switch orientation, swap the Width and Height values (portrait: height > width; landscape: width > height).
6. Click **OK** to apply.
7. When prompted, choose **Keep current size** or **Fit to page** for existing content.
