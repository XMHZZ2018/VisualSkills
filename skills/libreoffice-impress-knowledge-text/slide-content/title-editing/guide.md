# Edit Slide Title and Text Boxes (LibreOffice Impress 7.3.7)

## Add Title or Text to a Slide

- Click **Click to add Title** and type — Impress enters text editing mode automatically
- Click **Click to add Text** and type — same behavior
- Click outside the text box to deselect

## Create a New Text Box

1. Click **Insert Text Box** on the Standard or Drawing toolbar (or press **F2**)
2. Click on the slide for a single-line box, or click and drag to set a fixed width
3. Type or paste text
4. Click outside to deselect

## Move a Text Box

1. Click the text box border so selection handles appear
2. Hover over the border until the cursor changes to a move cursor (hand)
3. Click, hold, and drag to the new position; release

## Resize a Text Box

1. Click the text box so selection handles appear
2. Hover over a handle until the cursor changes shape
3. Drag the handle:
   - **Corner handles** — change width and height simultaneously
   - **Top/bottom handles** — change height only
   - **Left/right handles** — change width only
4. Release when at the required size

## Delete a Text Box or Element

1. Click the text box to select it
2. Press **Delete** (macOS: **Ctrl+D**) or **Backspace**

## Add a Border to a Text Box

**Via Line and Filling toolbar:**
1. Select the text box
2. Set **Line Style**, **Line Width**, and **Line Color** from the toolbar
3. Click outside to exit

**Via dialog:** Right-click the border → **Line**, or go to **Format > Object and Shape > Line**

**Via Sidebar:** **Properties** deck → **Line** panel

## Notes

- Use **Normal view** (not Master view) when modifying slide elements
- Changing a layout after content is added preserves content but may require reformatting
