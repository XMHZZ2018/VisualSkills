# Bold, Underline, and Strikethrough (LibreOffice Impress 7.3.7)

Apply bold, underline, or strikethrough directly to selected characters or words using direct/manual character formatting.

## Select Text First

- Click and drag over text to highlight it
- Double-click to select a word; triple-click to select a paragraph
- Click text, then hold **Shift** and use arrow keys to extend selection
- Click the text box border to select all text in the box

## Apply Formatting

Use any of these methods:

**Toolbar:** Click the formatting tool (Bold, Underline, Strikethrough) on the **Text Formatting** toolbar. Formatting is applied immediately.

**Menu:** Go to **Format > Character** on the Menu bar to open the Character dialog, apply formatting on the tabbed pages, then click **OK**.

**Right-click:** Right-click selected text and choose **Character** from the context menu, then use the Character dialog.

**Sidebar:** Open **Properties** deck on the Sidebar, click **Character** panel, and apply formatting. Changes are immediate.

## Notes

- Direct character formatting overrides any style-based formatting.
- Click outside the text box to deselect when done.
- For full details on Character dialog options, refer to the Writer Guide.
