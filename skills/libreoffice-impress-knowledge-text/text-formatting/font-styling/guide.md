# Font Styling

1. Select the text you want to format by clicking and dragging over it.
2. **Change font type**: In the formatting toolbar, click the font name dropdown and choose a font.
3. **Change font size**: Click the font size dropdown in the toolbar and enter or select a size.
4. **Bold**: Click the **B** button in the toolbar, or press `Ctrl+B`.
5. **Italic**: Click the **I** button in the toolbar, or press `Ctrl+I`.
6. **Underline**: Click the **U** button in the toolbar, or press `Ctrl+U`.
7. **Change font color (toolbar)**: Click the arrow next to the **A** (Font Color) icon in the toolbar, then select a color from the palette.
8. **Change font color (menu)**: Navigate to Format > Character > Font Effects tab, click the Font Color dropdown, select a color, then click OK.
9. **Access all font options at once**: Navigate to Format > Character > Fonts tab to set font type, size, and style in one dialog.
