# Add and Edit Text Boxes

1. Go to **Insert > Text Box** in the menu bar.
2. Click and drag on the slide to draw the text box at the desired size and position.
3. Type your content directly into the text box.
4. Click outside the text box to deselect it.
5. To edit existing text, double-click the text box to enter edit mode.
6. Select text by clicking and dragging, then apply formatting (bold, font size, color) via the toolbar.
7. To move a text box, single-click it to select (without entering edit mode), then drag it to a new position.
8. To resize, single-click the text box and drag any of the corner or edge handles.
9. To delete a text box, single-click to select it and press **Delete** or **Backspace**.
10. To group a text box with other objects, hold **Shift** and click each object, then right-click and select **Group > Group**.
