# Change Font Size (LibreOffice Impress 7.3.7)

## Quick Resize (Keyboard Shortcuts)

Select text first, then:
- **Increase Font Size**: `Ctrl+]` (macOS `⌘+]`)
- **Decrease Font Size**: `Ctrl+[` (macOS `⌘+[`)

The size change depends on the standard sizes available for the font in use.

## Using the Text Formatting Toolbar

1. Select the text to resize
2. Use the font size field on the **Text Formatting** toolbar to set a specific size

## Selecting Text

- **Entire text box**: Click the border of the text box (selection handles appear) — formatting applies to all text inside
- **Partial text**: Click and drag, double-click for a word, triple-click for a paragraph, or click then `Shift+arrow keys`

## Note on AutoLayout Text Boxes

AutoLayout text boxes automatically adjust font size to fit the content. When a long piece of text is inserted, the font size may decrease to fit the box.
