# Change Font Family / Typeface (LibreOffice Impress 7.3.7)

## Select Text First

- **Click the text box border** — selects all text in the box
- **Click and drag** — selects a range of characters
- **Double-click** — selects a word; **triple-click** — selects a paragraph

## Change Font Family

**Method 1 — Text Formatting Toolbar**
Use the font name drop-down on the Text Formatting toolbar. Changes apply immediately.

**Method 2 — Character Dialog**
1. Select the characters or words to format
2. Open the Character dialog via:
   - Right-click selected text → **Character**
   - **Format > Character** on the Menu bar
3. Go to the **Fonts** page and select the desired font family
4. Click **OK** to apply and close

**Method 3 — Sidebar**
1. Select the text box (selection handles visible)
2. On the Sidebar, click **Properties** → **Character** panel
3. Change the font family — applies immediately

## Apply to All Text in a Text Box

Click the **border** of the text box (not inside it) so selection handles appear, then change the font — formatting applies to all text in the box.

## Using Styles (for consistency across slides)

Font family can be set via **Presentation Styles** (used in AutoLayout text boxes) or **Drawing Styles** (used in text boxes and graphic objects). Modify a style to propagate the font change wherever that style is used.

- Go to **Format > Character** to set font within a style definition
- Note: Presentation Styles cannot be deleted or renamed; Drawing Styles can be modified or duplicated as custom styles
