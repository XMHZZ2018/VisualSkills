# Bullet and Numbered Lists

1. Click in a text box on your slide where you want to add a list.
2. To create a bullet list, click the **Toggle Unordered List** button in the formatting toolbar, or press **Shift+F12**.
3. To create a numbered list, click the **Toggle Ordered List** button in the formatting toolbar, or press **F12**.
4. To change the bullet style, click the dropdown arrow next to the bullet list button and select a style, or click **More Bullets** for additional options.
5. To change the numbering style, click the dropdown arrow next to the numbered list button and select a style.
6. Type your first list item, then press **Enter** to automatically create the next bullet or number.
7. To create a nested (indented) sub-item, press **Tab** at the beginning of a new line to indent it one level.
8. To move a nested item back to the parent level, press **Shift+Tab**.
9. To end the list and return to normal text, press **Enter** twice on the last empty list item.
10. To apply a list style to existing text, select the text first, then click the bullet or numbered list button.
