# Change Font Color (LibreOffice Impress 7.3.7)

Font color is a character formatting attribute applied directly to selected characters, words, or sentences.

## Select Text First

Before changing color, select the target text using one of these methods:

- Click and drag over the text to highlight it
- Double-click to select a word; triple-click to select a paragraph
- Click in text, then hold **Shift** and use arrow keys to extend selection
- Click the text box border to select **all** text in the box

## Apply Font Color

**Option 1 — Text Formatting toolbar**
Use the font color tool on the **Text Formatting toolbar**. Formatting is applied immediately.

**Option 2 — Character dialog**
1. Select the characters to recolor
2. Open the Character dialog using either:
   - Right-click selected text → **Character**
   - **Format > Character** on the Menu bar
3. Use the formatting options on the tabbed pages of the dialog
4. Click **OK** to apply and close
5. Click outside the text box to deselect

**Option 3 — Sidebar**
1. Select the text box so selection handles are visible
2. On the Sidebar, click **Properties** to open the Properties deck
3. Click **Character** to open the Character panel
4. Apply color using the available tools — formatting is immediate
5. Click **More Options** to open the full Character dialog if needed

## Notes

- Direct character formatting overrides any style-based formatting
- Click outside the text box when done to deselect
