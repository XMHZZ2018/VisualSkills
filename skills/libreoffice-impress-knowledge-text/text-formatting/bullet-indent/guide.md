# Bullets and Indentation (LibreOffice Impress 7.3.7)

## Creating a List

**In AutoLayout text boxes:** Click the placeholder text, start typing — lists are unordered by default.

**In regular text boxes:** First enable a list type via one of:
- **Text Formatting toolbar** → **Toggle Unordered List** or **Toggle Ordered List**
- **Sidebar** → **Properties** → **Lists panel** → **Toggle Unordered List** or **Toggle Ordered List**
- **Format > Lists > Unordered List** or **Ordered List**

Press **Enter** to create a new list point. Press **Shift+Enter** to add a new line within the same list point (same indentation).

## Adjusting Indent Level

**Increase indent (demote):**
- **Tab**
- **Alt+Shift+Right Arrow** (macOS: **⌥+Shift+Right Arrow**)
- **Outline toolbar** → **Demote**
- **Sidebar** → **Properties** → **Lists panel** → **Demote**
- **Format > Lists > Demote**

**Decrease indent (promote):**
- **Shift+Tab**
- **Alt+Shift+Left Arrow** (macOS: **⌥+Shift+Left Arrow**)
- **Outline toolbar** → **Promote**
- **Sidebar** → **Properties** → **Lists panel** → **Promote**
- **Format > Lists > Promote**

## Reordering List Points

**Move down** (without changing indent): **Alt+Shift+Down Arrow** (macOS: **⌥+Shift+Down Arrow**), or **Outline toolbar / Lists panel / Format > Lists** → **Move Down**

**Move up** (without changing indent): **Alt+Shift+Up Arrow** (macOS: **⌥+Shift+Up Arrow**), or **Outline toolbar / Lists panel / Format > Lists** → **Move Up**

## Changing List Appearance

- **Bullets and Numbering dialog** — change list type, properties, size, position, and scope for entire list or single point
- **Text Formatting toolbar** → dropdown arrow on **Toggle Unordered/Ordered List** → **More Bullets** or **More Numbering**
- **Sidebar** → **Properties** → **Lists panel**
