# Presenter View / Monitor Setup (LibreOffice Impress 7.3.7)

## Multiple Displays (Slide Show Settings)

1. Go to **Slide Show > Slide Show Settings** on the Menu bar.
2. Under **Multiple Displays**, select which display to use for full screen mode from the **Presentation display** drop-down list.

> Note: This setting is saved locally per computer, not with the presentation file. If opened on a different computer, that computer's local settings apply.

---

## Presenter Console

Requires dual displays (one can be a laptop). Provides the presenter with:
- Current slide (as seen by audience)
- Next slide preview
- Slide notes
- Presentation timer

### Views

| Control | Action |
|---|---|
| **Previous / Next** arrows | Navigate slides |
| **Notes** | Toggle notes view |
| **Slides** | Toggle slide thumbnail view |
| **Exchange** | Switch Presenter Console between displays |

> Note: The Presenter Console only works with operating systems that support two displays.

---

## Starting the Slide Show

- **F5** — Start from first slide
- **Shift+F5** — Start from current slide
- **Slide Show > Start from First Slide** or **Start from Current Slide**

### Navigating During the Show

- **Advance:** Left-click, Down arrow, Right arrow, Page Down, Spacebar
- **Go back:** Up arrow, Left arrow, Page Up
- **Right-click** to open context menu (Go to Slide, End Show, pen tools, etc.)
- **Esc** — Exit slide show at any time
