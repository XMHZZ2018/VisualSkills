# Slide Organization (LibreOffice Impress 7.3.7)

To insert a new slide, select a slide in the Slides panel and go to **Slide > New Slide** on the Menu bar, or just press *Ctrl+M*. The new slide appears right after the selected one, using the same master slide. If nothing is selected, it lands at the end.

**Duplicating** is handy when you want to keep the formatting, layout, and animations from an existing slide. Select your slide, then go to **Slide > Duplicate Slide**, or right-click and choose **Duplicate Slide** from the context menu. You can also click **Duplicate Slide** on the Presentation toolbar. This is great for splitting a crowded slide into two without rebuilding from scratch.

To **reorder slides**, switch to **Slide Sorter** view in the Workspace — just click on a slide and drag it to its new position. You can do the same thing directly in the Slides panel on the left by dragging thumbnails around.

If you need to select multiple slides for moving or copying, hold *Ctrl* (or *Cmd* on macOS) and click each one. For a contiguous range, click the first, then *Shift*-click the last.

**Renaming** helps keep things tidy. Right-click a slide and select **Rename Slide**, type the new name, and hit **OK**. Descriptive names matter especially when you use summary slides.

When a slide has too much content, use **Slide > Expand Slide** to split it. Impress creates new slides from each first-level outline point, promoting sub-points up one level automatically. Rearrange the resulting slides as needed.

Read the screenshot `fig01.png` in this directory — you will see the result of expanding a slide, with the original slide's outline points split into separate slides.

To build an **agenda or summary slide**, select the first slide you want included, then go to **Slide > Summary Slide**. Impress generates a new slide at the end with bullet points pulled from the titles of all slides in the presentation — a quick way to create a table of contents. Note: the selected slide must contain an outline for this to work.

Read the screenshot `fig02.png` in this directory — you will see a generated summary slide with bullet points listing all slide titles in the presentation.

You can also **pull slides from another file** via **Slide > Insert Slide from File**, browse to the source presentation, and pick individual slides from the dialog. To copy between open presentations, use **Slide Sorter** view in both windows — select, copy with *Ctrl+C*, then paste with *Ctrl+V* in the target. Holding *Ctrl* while dragging between presentations copies instead of moving.

Read the screenshot `fig03.png` in this directory — you will see the Insert Slides/Objects dialog showing a tree of slides and objects from the source file.

Finally, to **delete** a slide, select it and go to **Slide > Delete Slide**, right-click and choose **Delete Slide**, or simply press the *Delete* key.
