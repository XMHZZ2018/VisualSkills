# Exporting Presentations (LibreOffice Impress 7.3.7)

Impress can export presentations to PDF, HTML, and various other formats, making it easy to share your work with anyone regardless of what software they have installed.

## Quick PDF Export

If you just need a straightforward PDF with default settings, click **Export Directly as PDF** on the Standard toolbar, or go to **File > Export As > Export Directly as PDF**. Pick a save location, give it a name, and hit **Save** — done.

## PDF Export with Options

For more control over quality and content, go to **File > Export as PDF** to open the **PDF Options** dialog. On the **General** tab, you can choose a page range, set image compression (JPEG quality or lossless), and reduce image resolution to keep file size down. The **Structure** section lets you include outlines, notes pages, comments as PDF annotations, and even hidden slides. Once everything looks right, click **Export**, choose your destination, and save.

Read the screenshot `fig01.png` in this directory — you will see the PDF Options dialog showing the General tab with Range, Images compression, and Structure settings.

## HTML Export

You can turn your presentation into a set of web pages via **File > Export**. In the file browser, choose *HTML Document (Impress)* as the format and click **Save**. The HTML Export wizard walks you through picking a design, then a publication type — **Standard HTML format** gives one page per slide with navigation links, **Single-document HTML** puts everything on one page, and **WebCast** generates an ASP or Perl application. You'll also choose an image format (PNG, GIF, or JPG) and monitor resolution, optionally fill in title page info, select a button style for navigation, and pick a color scheme. Note that animation effects and slide transitions are not preserved in HTML output.

Read the screenshot `fig02.png` in this directory — you will see the HTML Export wizard's Publication Type page with options for Standard HTML format, Single-document HTML, and WebCast.

## Other Formats

For anything else, go to **File > Export**, pick your destination folder, and select the desired format from the drop-down list at the bottom of the file browser. Depending on the format, an options dialog may appear for additional settings — adjust as needed, then click **Save** followed by **OK**.
