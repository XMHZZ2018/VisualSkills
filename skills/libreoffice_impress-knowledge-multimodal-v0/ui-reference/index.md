# LibreOffice Impress 24.2 — UI Reference

LibreOffice Impress is a presentation application for creating, editing, and presenting slide decks. It supports ODP and PowerPoint formats, slide transitions, animations, and PDF export. The interface follows a traditional layout: menu bar, two toolbar rows, a left slide panel, a central canvas, and a right properties sidebar.

## Regions

| Region | Description |
|--------|-------------|
| [File Menu](regions/file-menu.md) | Document lifecycle: New, Open, Save, Export, Print, Properties. Includes PDF Options, Print, and Document Properties dialogs. |
| [Edit, View, Window & Help Menus](regions/standard-menus.md) | Clipboard, undo/redo, find/replace, view modes, toolbar toggles, grid/snap, sidebar panels, window management, and help resources. |
| [Insert Menu](regions/insert-menu.md) | Adding content: Image, Chart, Table, Shapes, Text Box, Hyperlink, Fields, Header/Footer, Form Controls. |
| [Format Menu](regions/format-menu.md) | Text styling, paragraph layout, bullets/numbering, object arrangement. Includes Character, Paragraph, and Bullets dialogs. |
| [Slide & Slide Show Menus](regions/slide-menus.md) | Slide management (add, duplicate, delete, layout, properties, masters, hide/show, reorder). Presentation playback (start, rehearse, custom shows, settings). |
| [Tools Menu](regions/tools-menu.md) | Spelling, language, macros, redaction, extensions, Customize dialog, and application-wide Options. |
| [Standard Toolbar](regions/standard-toolbar.md) | First icon row: New, Open, Save, PDF Export, Print, clipboard, undo/redo, search, spelling, grid toggles. |
| [Drawing Toolbar](regions/drawing-toolbar.md) | Second icon row: selection, colour pickers, line/rectangle/ellipse/arrow tools, shape palettes, rotate, align, arrange, shadow, crop, extrusion. |
| [Slides Panel & Canvas](regions/slides-panel-and-canvas.md) | Left thumbnail panel (navigation, context menu, drag-reorder) and central canvas (text placeholders, right-click menus, scrollbars). |
| [Properties Panel & Status Bar](regions/properties-and-status.md) | Right sidebar (slide format, orientation, background, master slide, layouts grid), sidebar icon strip (8 panels), and bottom status bar (slide position, zoom). |
