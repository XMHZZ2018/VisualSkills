# Text Box Formatting (LibreOffice Impress 7.3.7)

To insert a text box, click **Insert Text Box** on the Standard toolbar or the Drawing toolbar — or just press **F2** for a quick horizontal text box. Click on the slide to create a single-line box that grows as you type, or click and drag to set a fixed width with automatic line wrapping.

For font and text styling, select your text and use the **Text Formatting** toolbar that appears automatically. You can also go to **Format** on the Menu bar to access formatting options, or open the **Paragraph** and **Character** panels in the Properties deck on the Sidebar. Direct formatting overrides any applied style, so use it intentionally.

To add a border around a text box, click the box so its border is visible, then use the **Line and Filling** toolbar (enable it via **View > Toolbars > Line and Filling** if hidden). Pick a line style from **Line Style**, set the thickness in **Line Width**, and choose a color from **Line Color**. For more control — like corner rounding or transparency — right-click the border and select **Line**, or go to **Format > Object and Shape > Line**. On the **Line** tab, set *Style*, *Color*, *Width*, and *Transparency*. Under *Corner Style*, choose **Rounded** for softer edges — a width of at least 0.35cm makes rounded corners clearly visible.

Read the screenshot `fig01.png` in this directory — you will see the Line dialog showing Style, Color, Width, Transparency, and Corner Style options for text box borders.

You can also set borders through the Sidebar: click **Properties**, expand the **Line** panel, and adjust line style, width, color, and transparency there.

For background fills, expand the **Area** panel in the Properties Sidebar deck. This lets you apply a solid color, gradient, or other fill to your text box — handy for making code blocks or callout boxes stand out.

To precisely position or resize a text box, use **Format > Object and Shape > Position and Size** (or press **F4**). Set exact *Position X/Y* and *Width/Height* values. Check **Keep ratio** to lock proportions while resizing. The **Adapt** section offers **Fit width to text** and **Fit height to text** to auto-size the box to its content.

Read the screenshot `fig02.png` in this directory — you will see the Position and Size dialog with Position X/Y, Width/Height fields, Keep ratio checkbox, and Adapt options.

To rotate a text box, click its border twice (not a double-click) until the handles change to curved arrows, then drag a corner handle. For precise angles, open the **Position and Size** dialog, go to the **Rotation** tab, and enter a degree value in **Rotation Angle**.

Read the screenshot `fig03.png` in this directory — you will see a text box rotated at an angle with rotation handles visible.
