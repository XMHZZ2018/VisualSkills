# LibreOffice Impress Knowledge (multimodal-v0)

This skill bundles two complementary resources:

1. **Task guides** — step-by-step how-to guides for common presentation tasks (presentation setup, text & content, shapes & diagrams, images & objects, animations).
2. **UI reference** — a per-region catalog of menus, toolbars, dialogs, and panels with tightly-cropped screenshots, extracted automatically by the UI-explorer pipeline.

When you get a task, first check if a task guide matches. For fine-grained UI questions ("where is X?", "what does this dialog look like?", "what's on the drawing toolbar?"), consult the UI reference.
