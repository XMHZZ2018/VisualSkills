# Configuring Keyboard Shortcuts (OpenToonz 1.7)

OpenToonz ships with a full set of predefined keyboard shortcuts for tools, modifiers, and menu commands — but you're free to remap any of them or add new ones. Shortcuts are stored per user, so everyone on a shared machine can have their own layout without stepping on each other.

To set up or change a shortcut, go to **File > Configure Shortcuts…** to open the configuration window. You'll see a tree of folders (Tools, Tool Modifiers, menu commands, etc.) and a search field at the top. Find the command you want, select it, then simply press the key or key combination you'd like to assign. That's it — the binding is live immediately.

See `fig01.png`.

If you need to clear a binding, select the command in the same window and hit the **Remove** button next to the key field. The shortcut disappears and the command reverts to having no hotkey.

At the bottom of the dialog you'll find **Shortcut Presets**. You can **Load** a saved preset, **Save As** your current set under a new name, or **Delete** presets you no longer need. There's also an **Export Current Shortcuts** button if you want to back them up or share them with a colleague, and **Clear All Shortcuts** if you'd rather start from a blank slate.

One handy behavior worth knowing: if you **press and hold** a tool shortcut key (like **B** for Brush), you get a temporary switch — the tool activates only while the key is down, and releasing it snaps you back to whatever tool you were using before. Great for quick one-stroke corrections without disrupting your workflow.

The default tool keys are single letters — **A** (Animate), **S** (Selection), **B** (Brush), **G** (Geometric), **F** (Fill), **E** (Eraser), **T** (Tape), **K** (Style Picker), **V** (Skeleton), and so on. Navigation uses modifier combos: hold **Space** for the Hand tool, **Shift+Space** for Zoom, and **Ctrl/Cmd+Space** for Rotate. Playback is equally quick — **P** plays, **L** toggles loop, comma/period step through drawings, and adding **Shift** steps through frames instead.
