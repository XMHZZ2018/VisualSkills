# Tab Groups

1. Open Chrome with multiple tabs you want to organize.
2. Click a tab to select it. Hold **Ctrl** (Windows) or **Cmd** (Mac) and click to select individual tabs, or hold **Shift** and click to select a range.
3. Right-click one of the selected tabs and choose **Add tabs to new group**.
4. In the dialog, type a name for the group (e.g., "Slides", "Week 8").
5. Select a color to visually distinguish the group.
6. Click the colored group label in the tab bar to collapse or expand the group.
7. To add more tabs to an existing group, right-click a tab and select **Add tab to group**, then choose the target group.
8. To move a tab out of a group, right-click it and select **Remove from group**.
9. To rename or recolor a group, right-click the group label and select the desired option.
10. To close all tabs in a group, right-click the group label and select **Close group**.
