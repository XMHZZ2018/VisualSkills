# Move Tabs Between Windows

1. To move a tab to an existing window, click and hold the tab you want to move.
2. Drag the tab out of the current window until it becomes a floating thumbnail.
3. Drop the tab onto the tab bar of the destination window to merge it in.

**To move a tab to a new window:**

4. Right-click the tab you want to move.
5. Select **Move tab to new window** from the context menu.

**To drag multiple tabs at once:**

6. Hold `Ctrl` (Windows/Linux) or `Cmd` (Mac) and click each tab you want to select.
7. Click and drag the selected tabs out to a new or existing window.

**To merge all tabs from one window into another:**

8. Right-click any tab in the window you want to merge.
9. Select **Move all tabs to new window** or drag the entire group to the destination window's tab bar.
