# View Download History

1. Open Chrome and navigate to `chrome://downloads`
2. View your full download history — files are listed with name, date, and source URL
3. Click on a file name to open it if it still exists on disk
4. Click **Show in folder** to reveal the file in your system file manager
5. Click the **X** next to any entry to remove it from the download history
6. Use the search bar at the top to filter downloads by name

**To change the default download location:**

1. Click the three-dot menu (top right) and select **Settings**
2. In the left sidebar, click **Advanced** > **Downloads**
3. Click **Change** next to the current download location and select a new folder
4. Toggle **Ask where to save each file before downloading** if you want to choose a location per download

**To enable auto-open for a file type:**

1. Download a file of the desired type
2. Once complete, click the upward arrow next to the file in the download tray
3. Select **Always open files of this type**
