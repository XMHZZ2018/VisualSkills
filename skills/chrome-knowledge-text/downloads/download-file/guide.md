# Download a File

1. Navigate to the webpage containing the file you want to download.
2. Click the download link or button on the page.
3. Chrome will automatically begin downloading the file — a progress indicator appears in the top-right corner of the browser.
4. Click the download indicator (arrow icon) to view download progress.
5. Once complete, click **Open file** to open it, or **Show in folder** to locate it in your file system.
6. To manage all downloads, navigate to `chrome://downloads` or press **Ctrl+J** (Windows) / **Cmd+J** (Mac).
