# Manage Site Permissions

1. Open Chrome and navigate to `chrome://settings/privacy`
2. Click **Site Settings** under the Privacy and Security section
3. Select a permission type (e.g., **Camera**, **Microphone**, **Location**, **Notifications**)
4. Under the default behavior, select **Sites can ask to use [permission]** to allow prompts, or **Don't allow sites to use [permission]** to block globally
5. Scroll down to view **Allowed** and **Not allowed** site lists for that permission
6. To change a specific site's permission, click the site URL in either list
7. Use the dropdown next to the permission to set it to **Allow**, **Block**, or **Ask**
8. To remove a site from the list entirely, click the trash icon next to it
9. Navigate directly to a specific permission via `chrome://settings/content/camera`, `chrome://settings/content/microphone`, `chrome://settings/content/location`, or `chrome://settings/content/notifications`
