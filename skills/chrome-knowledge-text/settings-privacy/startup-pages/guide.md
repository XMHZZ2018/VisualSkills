# Configure Startup Pages

1. Open Chrome and click the three-dot menu (⋮) in the top-right corner.
2. Select **Settings**.
3. In the left sidebar, click **On startup**.
4. Choose one of the following options:
   - **Open the New Tab page** — starts Chrome with a blank new tab.
   - **Continue where you left off** — restores all tabs from the previous session.
   - **Open a specific page or set of pages** — launches one or more URLs on startup.
5. If you selected **Open a specific page or set of pages**:
   - Click **Use current pages** to add all currently open tabs, or
   - Click **Add a new page**, enter the URL, and click **Add**.
6. To remove a startup page, click the three-dot icon next to it and select **Remove**.

> Direct link: `chrome://settings/onStartup`
