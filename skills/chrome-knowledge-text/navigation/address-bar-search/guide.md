# Search from Address Bar

1. Click the address bar (Omnibox) at the top of Chrome.
2. Type a search query or URL and press **Enter**.
   - For searches, Chrome uses your default search engine.
   - For URLs, Chrome navigates directly to the site.

**If the search bar is missing from the New Tab page:**

1. Click the three-dot menu (top-right) and select **Settings**.
2. Navigate to **chrome://settings/search** or click **Search engine** in the left sidebar.
3. Under "Search engine used in the address bar", open the dropdown and select a different search engine (e.g., **Google**).
4. Open a new tab — the search bar will now appear.
