# Show Bookmark Bar

1. Click the three-dot menu (⋮) in the top-right corner of Chrome.
2. Hover over **Bookmarks**.
3. Click **Show bookmarks bar** to toggle it on (checkmark = visible).
4. To add the current page as a bookmark, press `Ctrl+D` (Windows) / `Cmd+D` (Mac), or go to **Bookmarks > Bookmark this tab**.
5. To bookmark all open tabs at once, go to **Bookmarks > Bookmark all tabs**.
6. To manage and organize bookmarks, go to **Bookmarks > Bookmark manager** (or press `Ctrl+Shift+O` / `Cmd+Shift+O`).
7. In Bookmark Manager, click the three-dot menu in the top-right and select **Add new folder** to create a folder.
8. Drag and drop bookmarks to reorder them or move them into folders.
9. To rename a bookmark or folder, right-click it and select **Edit** (bookmark) or **Rename** (folder).
10. To open all bookmarks in a folder simultaneously, right-click the folder and select **Open all**.
