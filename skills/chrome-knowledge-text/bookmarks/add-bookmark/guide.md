# Add a Bookmark

1. Navigate to the page you want to bookmark.
2. Click the **star icon** in the address bar, or press `Ctrl+D` (Windows/Linux) / `Cmd+D` (Mac).
3. Edit the bookmark name if desired.
4. Select a folder destination from the dropdown (e.g., Bookmarks Bar).
5. Click **Done** to save the bookmark.

**To bookmark all open tabs at once:**

1. Click the **three-dot menu** (⋮) in the top-right corner.
2. Select **Bookmarks > Bookmark all tabs** (or press `Ctrl+Shift+D` / `Cmd+Shift+D`).
3. Choose a folder and click **Save**.

**To show the Bookmarks Bar:**

1. Click the **three-dot menu** and select **Bookmarks > Show bookmarks bar**, or press `Ctrl+Shift+B` / `Cmd+Shift+B`.
