# Manage Bookmarks

1. Open the Bookmark Manager via `chrome://bookmarks` or go to **Menu > Bookmarks > Bookmark manager**.
2. Drag and drop bookmarks to reorder them — place most-used sites in the first positions.
3. To create a folder, click the **three-dot menu** (top right) and select **Add new folder**, then name it.
4. To rename a bookmark or folder, right-click it and select **Edit** (for bookmarks) or **Rename** (for folders).
5. To move a bookmark into a folder, drag it onto the target folder.
6. To delete a bookmark or folder, right-click it and select **Delete**.
7. To bookmark all open tabs at once, go to **Menu > Bookmarks > Bookmark all tabs** (or press `Ctrl+Shift+D` / `Cmd+Shift+D`).
8. To open all bookmarks in a folder at once, right-click the folder and select **Open all**.
9. To show or hide the bookmarks bar, go to **Menu > Bookmarks > Show bookmarks bar** (or press `Ctrl+Shift+B` / `Cmd+Shift+B`).
10. To shorten bookmark names and save bar space, right-click a bookmark, select **Edit**, and shorten or clear the name field, then click **Save**.
